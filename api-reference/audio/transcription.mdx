## Overview

Transcribe audio into text using high-performance models. This endpoint requires `multipart/form-data` encoding to handle audio file uploads.

### Important Notes
- **Max File Size**: 25MB
- **Supported Formats**: `mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `wav`, `webm`
- **Billing**: Calculated per second of audio duration.
