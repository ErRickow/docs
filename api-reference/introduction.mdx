## Base URL

The base URL for all requests to the Neosantara API is:

```plaintext with prefix /v1
https://api.neosantara.xyz/v1
```

## Authentication

All Neosantara endpoints are authenticated using API keys.
[Get your free API key](https://app.neosantara.xyz/api-keys).

```bash icon="terminal"
curl "https://api.neosantara.xyz/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <NAI_API_KEY>" \
  -d '{
    "model": "nusantara-base",
    "messages": [
      {
        "role": "user",
        "content": "Ceritakan sebuah kisah singkat tentang seekor kancil yang cerdik."
      }
    ],
    "stream": false
  }'
```
