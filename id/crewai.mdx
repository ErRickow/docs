CrewAI adalah framework tingkat produksi sumber terbuka untuk mengorkestrasi sistem agen AI. Ini memungkinkan banyak agen AI untuk berkolaborasi secara efektif dengan mengasumsikan peran dan bekerja menuju tujuan bersama. Framework ini mendukung otomatisasi sederhana dan aplikasi kompleks yang memerlukan perilaku agen yang terkoordinasi.

## Menginstal Perpustakaan

```bash icon="terminal"
uv pip install crewai
```

Atur kunci API Neosantara kamu:

```bash icon="terminal"
export NAI_API_KEY=***
```

## Contoh

```python icon="python"
import os
from crewai import LLM, Task, Agent, Crew

llm = LLM(model="openai/nusantara-base",
          api_key=os.environ.get("NAI_API_KEY"),
          base_url="https://api.neosantara.xyz/v1"
        )

research_agent = Agent(
    llm = llm,
    role="Research Analyst",
    goal="Temukan dan rangkum informasi tentang topik tertentu",
    backstory="kamu adalah peneliti berpengalaman dengan perhatian terhadap detail",
    verbose=True  # Aktifkan pembuatan log untuk debugging
)

research_task = Task(
    description="Lakukan penelitian menyeluruh tentang Agen AI.",
    expected_output="Daftar dengan 10 poin bulet informasi paling relevan tentang Agen AI",
    agent=research_agent
)

# Jalankan crew
crew = Crew(
    agents=[research_agent],
    tasks=[research_task],
    verbose=True
)

result = crew.kickoff()

# Mengakses output tugas
task_output = research_task.output
print(task_output)
```
