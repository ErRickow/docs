CrewAI adalah framework tingkat produksi open-source untuk mengorkestrasi sistem agen AI. Framework ini memungkinkan beberapa agen AI untuk berkolaborasi secara efektif dengan mengambil peran dan bekerja menuju tujuan bersama. Framework ini mendukung otomatisasi sederhana maupun aplikasi kompleks yang memerlukan perilaku agen yang terkoordinasi.

## Instalasi Library

```bash icon="terminal"
pip install crewai
```

Atur API Key Neosantara Kamu:

```bash icon="terminal"
export NAI_API_KEY=***
```

## Contoh Penggunaan

```python icon="python"
import os
from crewai import LLM, Task, Agent, Crew

llm = LLM(model="openai/nusantara-base",
          api_key=os.environ.get("NAI_API_KEY"),
          base_url="https://api.neosantara.xyz/v1"
        )

research_agent = Agent(
    llm = llm,
    role="Research Analyst",
    goal="Temukan dan ringkas informasi tentang topik tertentu",
    backstory="Kamu adalah peneliti berpengalaman dengan perhatian terhadap detail",
    verbose=True  # Aktifkan logging untuk debugging
)

research_task = Task(
    description="Lakukan riset mendalam tentang AI Agents.",
    expected_output="Daftar dengan 10 poin informasi paling relevan tentang AI Agents",
    agent=research_agent
)

# Eksekusi crew
crew = Crew(
    agents=[research_agent],
    tasks=[research_task],
    verbose=True
)

result = crew.kickoff()

# Mengakses output tugas
task_output = research_task.output
print(task_output)
```

* AI Agents adalah program komputer yang menggunakan kecerdasan buatan (AI) untuk melakukan tugas-tugas yang biasanya memerlukan kecerdasan manusia, seperti penalaran, pemecahan masalah, dan pengambilan keputusan. Mereka dapat digunakan dalam berbagai aplikasi, termasuk asisten virtual, chatbot layanan pelanggan, dan kendaraan otonom.
* Ada beberapa jenis AI Agents, termasuk simple reflex agents, model-based reflex agents, goal-based agents, utility-based agents, dan learning agents. Setiap jenis agen memiliki kekuatan dan kelemahannya sendiri, dan cocok untuk tugas dan lingkungan tertentu.
* Simple reflex agents adalah jenis AI Agent yang paling dasar, dan bereaksi terhadap keadaan lingkungan saat ini tanpa mempertimbangkan konsekuensi di masa depan. Mereka sering digunakan dalam aplikasi di mana lingkungan dapat diamati sepenuhnya, seperti dalam permainan.
* Model-based reflex agents, di sisi lain, mempertahankan model internal lingkungan dan menggunakan model ini untuk membuat keputusan. Mereka lebih canggih daripada simple reflex agents dan dapat menangani lingkungan yang hanya dapat diamati sebagian.
* Goal-based agents dirancang untuk mencapai tujuan tertentu, dan menggunakan berbagai teknik seperti perencanaan dan pengambilan keputusan untuk mencapai tujuan tersebut. Mereka sering digunakan dalam aplikasi seperti robotika dan kendaraan otonom.
* Utility-based agents membuat keputusan berdasarkan fungsi utilitas yang memperkirakan keinginan dari setiap tindakan yang mungkin. Mereka sering digunakan dalam aplikasi seperti sistem rekomendasi dan sistem pendukung keputusan.
* Learning agents menggunakan algoritma pembelajaran mesin untuk meningkatkan kinerja mereka seiring waktu. Mereka dapat belajar dari pengalaman, dan beradaptasi dengan perubahan lingkungan dan tugas.
* AI Agents dapat diklasifikasikan ke dalam dua kategori: agen otonom dan agen semi-otonom. Agen otonom beroperasi secara independen, sementara agen semi-otonom memerlukan intervensi manusia untuk berfungsi.
* Keuntungan dari AI Agents termasuk peningkatan efisiensi, peningkatan akurasi, dan peningkatan kemampuan pengambilan keputusan. Mereka juga dapat beroperasi 24/7 tanpa kelelahan, dan dapat menangani data dalam jumlah besar dan tugas yang kompleks.
* Pengembangan AI Agents melibatkan beberapa tantangan, termasuk memastikan keamanan, keselamatan, dan transparansi mereka. Ada juga kekhawatiran etis seputar penggunaan AI Agents, seperti bias, privasi, and akuntabilitas. Karena AI Agents menjadi lebih luas, penting untuk mengatasi tantangan ini dan memastikan bahwa mereka dikembangkan dan digunakan secara bertanggung jawab.
