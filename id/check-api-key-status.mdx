<Card icon="shield-check" title="Verifikasi akses Kamu" horizontal>
Gunakan endpoint ini untuk memvalidasi Kunci API Kamu, memeriksa **Tingkat Layanan** langganan Kamu saat ini, serta memantau batas penggunaan token **Harian** dan **Bulanan** Kamu.
</Card>

## Mengembalikan

<ResponseField name="object" type="string">
Selalu "authorization".
</ResponseField>

<ResponseField name="id" type="string">
Pengenal unik untuk pemeriksaan otorisasi ini (format: `auth-` + 16 karakter hex).
</ResponseField>

<ResponseField name="authorized" type="boolean">
Menunjukkan apakah Kunci API yang diberikan valid dan aktif.
</ResponseField>

<ResponseField name="tier" type="string">
Tingkat layanan saat ini dari Kunci API (misalnya, "Free", "Developer", "Production").
</ResponseField>

<ResponseField name="permissions" type="array">
Daftar kemampuan yang diberikan kepada kunci ini (misalnya, `chat.completions`, `embeddings`).
</ResponseField>

<ResponseField name="usage_info" type="object">
Rincian mendalam tentang penggunaan token dan batasan.

<Expandable title="properti usage_info" defaultOpen>
<ResponseField name="daily" type="object">
<Expandable title="properti daily">
<ResponseField name="used" type="integer">Token yang digunakan hari ini.</ResponseField>
<ResponseField name="limit" type="integer">Batas token harian.</ResponseField>
<ResponseField name="remaining" type="integer">Sisa token untuk hari ini.</ResponseField>
<ResponseField name="reset_at" type="string">Waktu pengaturan ulang harian berikutnya.</ResponseField>
</Expandable>
</ResponseField>

<ResponseField name="monthly" type="object">
  <Expandable title="properti monthly">
    <ResponseField name="used" type="integer">Token yang digunakan bulan ini.</ResponseField>
    <ResponseField name="limit" type="integer">Batas token bulanan.</ResponseField>
    <ResponseField name="remaining" type="integer">Sisa token untuk bulan ini.</ResponseField>
    <ResponseField name="reset_at" type="string">Waktu pengaturan ulang bulanan berikutnya.</ResponseField>
  </Expandable>
</ResponseField>

</Expandable>
</ResponseField>

<ResponseField name="organization" type="string">
Pengenal organisasi (default: `org-neosantara-ai`).
</ResponseField>

<ResponseField name="created" type="integer">
Timestamp Unix saat pemeriksaan dilakukan.
</ResponseField>

<ResponseField name="_metadata" type="object">
Informasi status dan debugging internal.
<Expandable title="properti _metadata">
<ResponseField name="status" type="boolean">Pemeriksaan status sistem.</ResponseField>
<ResponseField name="processing_time" type="integer">Waktu yang dibutuhkan untuk memproses permintaan (ms).</ResponseField>
<ResponseField name="timestamp" type="string">Timestamp ISO dari permintaan.</ResponseField>
</Expandable>
</ResponseField>
