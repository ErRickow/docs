API Neosantara mendukung metode autentikasi standar di seluruh endpoint.

## Metode

- Bearer Token (Kunci API di header Authorization)
- Header Kunci API (misalnya, `x-api-key`)
- Variabel lingkungan (direkomendasikan untuk pengembangan lokal)

## Cara Menggunakan

- Atur header Authorization: `Authorization: Bearer KUNCI_API_NEOSANTARA_KAMU`
- Atau berikan kunci API melalui header seperti `x-api-key: <KUNCI>` tergantung pada pustaka klien.
- Untuk aplikasi berbasis browser, gunakan backend yang aman untuk memproksi permintaan dan menjaga kerahasiaan kunci.

## Contoh

Python

```python icon="python"
import requests
headers = {"Authorization": "Bearer KUNCI_API_NEOSANTARA_KAMU"}
payload = {"model": "nusantara-base", "messages": [{"role": "user", "content": "Halo"}]}
resp = requests.post("https://api.neosantara.xyz/v1/chat/completions", json=payload, headers=headers)
```

JavaScript

```js
const payload = {
  model: "nusantara-base",
  messages: [{ role: "user", content: "Halo" }]
};

const res = await fetch("https://api.neosantara.xyz/v1/chat/completions", {
  method: 'POST',
  headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer KUNCI_API_NEOSANTARA_KAMU' },
  body: JSON.stringify(payload)
})
```
