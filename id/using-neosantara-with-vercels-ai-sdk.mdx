Vercel AI SDK adalah pustaka Typescript yang kuat yang dirancang untuk membantu pengembang membangun aplikasi bertenaga AI. Dengan menggunakan Neosantara dan Vercel AI SDK, Kamu dapat dengan mudah mengintegrasikan AI ke dalam proyek TypeScript, React, atau Next.js Kamu. Dalam tutorial ini, kita akan melihat betapa mudahnya menggunakan model Neosantara dan Vercel AI SDK.

## Mulai Cepat: 15 baris kode

1. Instal Vercel AI SDK dan paket Vercel dari OpenAI.

```bash icon="terminal"
npm i ai @ai-sdk/openai-compatible
```

```bash icon="terminal"
yarn add ai @ai-sdk/openai-compatible
```

2. Buat instans klien Neosantara dan panggil fungsi `generateText` dengan nusantara-base untuk menghasilkan beberapa teks.

```javascript icon="js"
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { generateText } from 'ai';

const neosantara = createOpenAICompatible({
  name: 'neosantara_xyz',
  apiKey: process.env.NAI_API_KEY ?? "",
  baseURL: 'https://api.neosantara.xyz/v1',
});

const { text } = await generateText({
  model: neosantara('nusantara-base'),
  prompt: 'Tulis cerita tentang kucing yang menginginkan whiskas',
});

console.log(text)
```

## Streaming dengan Vercel AI SDK

Untuk melakukan streaming dari model Neosantara menggunakan Vercel AI SDK, cukup gunakan `streamText` seperti yang terlihat di bawah ini.

```javascript icon="js"
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { streamText } from "ai";

const neosantara = createOpenAICompatible({
  apiKey: process.env.NAI_API_KEY ?? "",
  baseURL: "https://api.neosantara.xyz/v1",
});

async function main() {
  const result = await streamText({
    model: neosantara("nusantara-base"),
    prompt: "Buat hari libur baru dan jelaskan tradisinya.",
  });

  for await (const textPart of result.textStream) {
    process.stdout.write(textPart);
  }
}

main();
```
