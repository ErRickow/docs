Neosantara memakai billing pay-as-you-go. Biaya request dipotong dari saldo kredit berdasarkan model yang dipilih, jenis request, dan unit penggunaan seperti token, gambar yang dibuat, durasi audio, atau durasi video.

<Note>
  Harga per model bersumber dari katalog model live. Buka [Ikhtisar Model](/id/models-overview) atau halaman model generated untuk melihat unit harga terbaru.
</Note>

## Memulai

1. Buat akun di [aplikasi Neosantara](https://app.neosantara.xyz).
2. Verifikasi email untuk menerima benefit kredit akun baru saat tersedia.
3. Buat API key dari [API Keys](https://app.neosantara.xyz/api-keys).
4. Pantau saldo dan penggunaan dari [halaman Usage](https://app.neosantara.xyz/usage).

## Cara billing bekerja

| Jenis penggunaan | Unit billing | Tempat mengecek |
| --- | --- | --- |
| Text, chat, reasoning, tools | Token input dan output | Halaman model generated dan metadata usage di response |
| Embeddings | Token input | [Model embeddings](/models/nusa-embedding-0001) |
| Image generation | Ukuran gambar atau unit generasi | Halaman model image generated |
| Audio transcription | Durasi audio | Halaman model audio generated |
| Video generation | Durasi atau unit provider | Halaman model video generated |

Sebagian model memakai harga IDR, sementara model yang mengikuti harga provider dapat ditampilkan dalam USD dan dikonversi saat billing.

## Diskon batch

Batch API dirancang untuk pekerjaan asynchronous volume tinggi yang tidak membutuhkan response langsung.

| Jenis penggunaan | Penyesuaian harga |
| --- | --- |
| Batch input | 50% dari harga input standar |
| Batch output | 50% dari harga output standar |

<Note>
  Diskon diterapkan otomatis untuk request yang dikirim melalui `/v1/batches`.
</Note>

## Cache read

Untuk model yang mendukung prompt caching, input yang dibaca dari cache dapat ditagih dengan harga cache-read terpisah. Jika halaman model tidak mencantumkan harga cache-read, harga input standar berlaku.

## Penggunaan tools

Definisi tool, panggilan tool, dan hasil tool ditagih sebagai token input dan output normal untuk model yang dipilih. Neosantara tidak menambahkan biaya orkestrasi terpisah untuk tool calling.

## Tier throughput

Batas throughput mengikuti tier akun. Upgrade tier diterapkan otomatis berdasarkan riwayat total deposit.

| Tier | Min. deposit kumulatif | RPM | ITPM | OTPM |
| --- | --- | --- | --- | --- |
| Free | Rp 0 | 3 | 15.000 | 2.000 |
| Basic | Rp 85.000 | 50 | 50.000 | 10.000 |
| Standard | Rp 670.000 | 1.000 | 450.000 | 90.000 |
| Pro | Rp 3.350.000 | 2.000 | 1.000.000 | 200.000 |
| Enterprise | Rp 6.700.000 | 4.000 | 4.000.000 | 800.000 |

Lihat [Rate Limits](/id/about/rate-limits) untuk header response, retry behavior, dan penanganan limit.

## Pengelolaan saldo

Saldo dan penggunaan saat ini tersedia di dashboard. Response API juga menyertakan metadata usage dan billing saat didukung.

Untuk menambah saldo, buka [halaman Billing](https://app.neosantara.xyz/billing), pilih **Top Up Balance**, lalu selesaikan pembayaran melalui metode yang tersedia.
