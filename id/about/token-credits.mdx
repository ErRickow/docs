## Biaya Kredit API

Penggunaan API Neosantara AI dihitung berdasarkan jumlah Token yang diproses per permintaan, baik untuk `input` maupun `output`. Setiap model memiliki karakteristik dan biaya Token yang berbeda.

### Cara Perhitungan Biaya Token

* **Pemrosesan Teks Dasar (misalnya, Chat Completions):**
  * Setiap permintaan teks dasar mengonsumsi Token berdasarkan panjang teks input (prompt dan riwayat obrolan) serta teks output yang dihasilkan.

* **Fungsi Khusus & Kemampuan Lanjutan (misalnya, function_calling, web_search):**
  * Ketika model menggunakan kemampuan lanjutan seperti `memanggil fungsi eksternal` atau melakukan `pencarian web`, biaya Token untuk permintaan tersebut akan mencerminkan kompleksitas komputasi tambahan dan pemrosesan data yang terlibat. Biaya pastinya bervariasi per permintaan berdasarkan operasi aktual yang dilakukan oleh model.

### Melacak Penggunaan Anda

Cara paling akurat untuk melacak biaya Token setiap permintaan, terutama untuk fitur lanjutan dan input multimodal, adalah melalui kolom `tokens_used` di bagian `_metadata` respons API atau dengan memeriksa [Dasbor Penggunaan](https://app.neosantara.xyz/usage).
