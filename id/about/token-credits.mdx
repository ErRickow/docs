### Cara Perhitungan Biaya

$$Harga = (\text{Token Input} \times \text{Harga Input}) + (\text{Token Output} \times \text{Harga Output})$$

**Contoh:**
Menggunakan `nusantara-base` (Input: Rp 300/1Jt, Output: Rp 1.500/1Jt) untuk permintaan dengan 500 token input dan 200 token output:

1.  Biaya Input: $500 / 1.000.000 \times 300 = \text{Rp } 0,15$
2.  Biaya Output: $200 / 1.000.000 \times 1.500 = \text{Rp } 0,30$
3.  **Total:** Rp 0,45
