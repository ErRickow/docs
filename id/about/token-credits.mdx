## Sistem Pay-As-You-Go

Neosantara menggunakan sistem penagihan **Pay-As-You-Go**. Biaya dihitung berdasarkan penggunaan riil (token atau gambar) tanpa biaya langganan bulanan wajib.

### Memulai

1. **Registrasi:** Buat akun baru di platform.
2. **Verifikasi Email:** Verifikasi email Kamu untuk mendapatkan **Kredit Saldo Awal Rp 20.000**.
3. **Akses API:** Akses diberikan segera setelah verifikasi, dengan penggunaan yang didebit dari saldo kredit Kamu.

---

## Struktur Harga

Biaya dihitung per permintaan dalam mata uang **Rupiah (IDR)**. Model dengan harga asli USD dikonversi otomatis menggunakan kurs dinamis.

### Batch Processing

Gunakan Batch API untuk pemrosesan asinkron dengan volume tinggi dengan tarif lebih rendah.

| Jenis Penggunaan | Penyesuaian Tarif |
| :--- | :--- |
| Batch Input | **50% dari** harga standar |
| Batch Output | **50% dari** harga standar |
