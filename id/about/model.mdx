## Panduan Pemilihan Model

Gunakan panduan ini untuk memilih model yang tepat untuk kebutuhan Anda:

1. **Konten Indonesia & Konteks Lokal**:
    - Prioritas: nusantara-base, archipelago-7b, bahasa-llm

2. **Pemrosesan Cepat & Efisien**:
    - Prioritas: bahasa-llm, gemini-1.5-flash, gemini-2.0-flash, gemma2-9b-it

3. **Pencarian Informasi & Analisis Mendalam**:
    - Prioritas: command-r-plus (untuk pencarian web), gemini-1.5-pro (untuk analisis)

4. **Panggilan Fungsi**:
    - Prioritas: nusantara-base, bahasa-llm, gemini-1.5-flash, gemini-1.5-pro, gpt-3.5-turbo, gpt-4o-mini, firefunction-v1, llama-3.3-70b-instruct

5. **Pembuatan Gambar**:
    - Gunakan: neosantara-gen-2045, replicate-stable-diffusion-xl

6. **Analisis Gambar (Visi)**:
    - Gunakan: vision-emas-2045

7. **Embedding (Pemetaan Teks)**:
    - Gunakan: nusa-embedding-0001, together-embed-v2-L

## Kemampuan Model

Berikut daftar kemampuan utama yang didukung oleh model Neosantara AI:

| Kemampuan | Deskripsi |
|------------|-------------|
| function_calling | Kemampuan model untuk memanggil fungsi eksternal |
| json_mode | Kemampuan model untuk menghasilkan output dalam format JSON terstruktur |
| web_search | Kemampuan model untuk mencari informasi dari web |
| streaming | Mendukung respons streaming (output bertahap) |
| image_generation | Kemampuan model untuk membuat gambar dari perintah teks |
| vision | Kemampuan model untuk menganalisis dan memahami konten dari gambar |
| embedding | Kemampuan model untuk mengubah teks menjadi representasi vektor numerik |
