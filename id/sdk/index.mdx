[AI Gateway Neosantara](/id/intro) menyediakan API drop-in yang kompatibel sehingga Kamu dapat beralih hanya dengan mengubah base URL. Gunakan SDK dan alat yang sama yang sudah Kamu kenal, dengan akses ke 100+ model dari setiap penyedia utama.

## Mulai cepat

Arahkan SDK Kamu yang sudah ada ke gateway:

<CodeGroup>

```javascript OpenAI SDK
import OpenAI from 'openai';
 
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});
 
const response = await client.chat.completions.create({
  model: 'grok-4.1-fast-non-reasoning', // Model apa pun yang tersedia
  messages: [{ role: 'user', content: 'Halo!' }],
});
```

```javascript Anthropic SDK
import Anthropic from '@anthropic-ai/sdk';
 
const client = new Anthropic({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/anthropic',
});
 
const message = await client.messages.create({
  model: 'grok-4.1-fast-non-reasoning',
  max_tokens: 1024,
  messages: [{ role: 'user', content: 'Halo!' }],
});
```

```bash cURL
curl https://api.neosantara.xyz/v1/chat/completions \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "grok-4.1-fast-non-reasoning",
    "messages": [{"role": "user", "content": "Halo!"}]
  }'
```

</CodeGroup>

## Mengapa menggunakan API ini?

*   **Tanpa vendor lock-in**: Beralih antara Claude, GPT, Gemini, dan model lainnya tanpa mengubah kode Kamu.
*   **Penagihan terpadu**: Satu saldo kredit untuk semua model alih-alih mengelola banyak akun dan tagihan.
*   **Fallback bawaan**: Keandalan yang ditingkatkan dengan perutean multi-penyedia dan optimalisasi regional.
*   **Dukungan streaming**: Respons waktu nyata dengan SSE di semua endpoint yang kompatibel.
*   **Paritas fitur lengkap**: Pemanggilan tool, output terstruktur, visi, dan embedding berfungsi persis seperti yang didokumentasikan.

## Memilih API

**Sudah menggunakan OpenAI?** Gunakan [API Kompatibel OpenAI](/id/sdk/openai-compat). Ubah base URL Kamu dan Kamu selesai.

**Menggunakan Claude Code atau SDK Anthropic?** Gunakan [API Kompatibel Anthropic](/id/sdk/anthropic-compat) untuk dukungan fitur native.

**Memulai baru?** Kami sangat merekomendasikan [API Responses](/id/sdk/responses-api) untuk antarmuka modern dan stateful, atau [Vercel AI SDK](/id/using-neosantara-with-vercels-ai-sdk) untuk pengalaman TypeScript terbaik.
