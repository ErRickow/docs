Neosantara AI menyediakan beberapa lapisan kompatibilitas, memungkinkan Kamu menggunakan SDK resmi dari penyedia AI populer. Pilih dokumentasi berdasarkan ekosistem yang Kamu gunakan.

  </Card>

## Mana yang Harus Kamu Pilih?

- **Kompatibel OpenAI**: Terbaik jika Kamu sudah memiliki aplikasi berbasis OpenAI atau menggunakan library universal seperti LangChain atau LlamaIndex.
- **Kompatibel Anthropic**: Ideal jika Kamu membangun dengan alat khusus Claude seperti Claude Code atau lebih menyukai struktur Anthropic Messages.
