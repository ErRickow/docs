Neosantara [AI Gateway](/id/intro) menyediakan API yang kompatibel dengan berbagai standar industri, memungkinkan Anda beralih antar model hanya dengan mengubah base URL. Gunakan SDK dan alat yang sudah Anda kuasai untuk mengakses 100+ model dari berbagai provider utama.

## [Quick Start](#quick-start)

Hubungkan SDK Anda yang sudah ada ke gateway Neosantara:

<CodeGroup>

```javascript OpenAI SDK
import OpenAI from 'openai';
 
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});
 
const response = await client.chat.completions.create({
  model: 'grok-4.1-fast-non-reasoning',
  messages: [{ role: 'user', content: 'Halo!' }],
});
```

```javascript Anthropic SDK
import Anthropic from '@anthropic-ai/sdk';
 
const client = new Anthropic({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/anthropic',
});
 
const message = await client.messages.create({
  model: 'claude-4.5-sonnet',
  max_tokens: 1024,
  messages: [{ role: 'user', content: 'Halo!' }],
});
```

</CodeGroup>

## [Mengapa Menggunakan API Ini?](#why-use-these-apis)

*   **Tanpa Vendor Lock-in**: Beralih antara Claude, GPT, Gemini, dan model lainnya tanpa mengubah kode Anda.
*   **Penagihan Terpadu**: Satu saldo kredit untuk semua provider, tidak perlu mengelola banyak akun dan faktur.
*   **Keandalan**: Dukungan redundansi multi-provider memastikan layanan tetap aktif meski satu provider sedang down.
*   **Fitur Lengkap**: Pemanggilan tool, output terstruktur, visi, dan embedding bekerja persis seperti yang didokumentasikan.

## [Memilih API](#choosing-an-api)

**Sudah menggunakan OpenAI?** Gunakan [API Kompatibel OpenAI](/id/sdk/openai-compat). Cukup ubah base URL Anda.

**Menggunakan Claude Code atau SDK Anthropic?** Gunakan [API Kompatibel Anthropic](/id/sdk/anthropic-compat) untuk dukungan fitur native.

**Memulai dari awal?** Kami sangat merekomendasikan [Responses API](/id/sdk/responses-api) untuk antarmuka modern dan stateful, atau [Vercel AI SDK](/id/using-neosantara-with-vercels-ai-sdk) untuk pengalaman TypeScript terbaik.
