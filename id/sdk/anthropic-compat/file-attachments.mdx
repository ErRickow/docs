# Lampiran File

Neosantara AI mendukung pengiriman gambar dan dokumen PDF secara native melalui API yang kompatibel dengan Anthropic. Ini memungkinkan Kamu membangun aplikasi multimodal.

### [Jenis File yang Didukung](#jenis-file-yang-didukung)

*   **Gambar**: `image/jpeg`, `image/png`, `image/gif`, `image/webp`
*   **Dokumen**: `application/pdf`

### [Contoh Permintaan](#contoh-permintaan)

```typescript
const message = await client.messages.create({
  model: 'claude-4.5-sonnet',
  max_tokens: 1024,
  messages: [{
    role: 'user',
    content: [
      {
        type: 'document',
        source: {
          type: 'base64',
          media_type: 'application/pdf',
          data: pdfBase64 // String base64 dari file PDF
        }
      },
      { type: 'text', text: 'Ringkas isi dokumen ini.' }
    ]
  }]
});
```
