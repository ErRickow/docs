description: "Gunakan SDK Anthropic resmi untuk mengakses Neosantara AI."
---

Neosantara [AI Gateway](/id/intro) menyediakan endpoint yang kompatibel dengan Anthropic Messages API, memungkinkan Kamu menggunakan SDK Anthropic atau alat seperti [Claude Code](https://code.claude.com) dengan mudah.

## [Base URL](#base-url)

`https://api.neosantara.xyz/anthropic`

<Warning>
Pastikan Kamu menggunakan sub-path `/anthropic` saat mengonfigurasi SDK Anthropic. Penggunaan root `/v1` akan menyebabkan kegagalan skema.
</Warning>

## [Quick Start SDK](#quick-start)

```typescript
import Anthropic from '@anthropic-ai/sdk';
 
const client = new Anthropic({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/anthropic',
});
 
const message = await client.messages.create({
  model: 'claude-4.5-sonnet',
  max_tokens: 1024,
  messages: [{ role: 'user', content: 'Halo, apa kabar?' }],
});
```
