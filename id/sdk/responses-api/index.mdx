API Neosantara mendukung **API Responses**, sebuah spesifikasi modern untuk interaksi model AI. API ini menyediakan antarmuka terpadu di seluruh penyedia dengan dukungan bawaan untuk percakapan stateful, streaming, pemanggilan tool, penalaran, dan input multi-modal.

Untuk perbandingan mendalam dengan API Chat Completions yang lama, lihat [Responses vs. Chat Completions](/id/response).

## Base URL

API Responses tersedia di:

`https://api.neosantara.xyz/v1`

## Autentikasi

API Responses mendukung autentikasi token Bearer standar:

*   **Kunci API**: Gunakan kunci API AI Neosantara Kamu dengan header `Authorization: Bearer <token>`.

## Fitur yang Didukung

API Responses dirancang untuk membangun aplikasi AI tingkat lanjut:

*   **[Generasi Teks](/id/sdk/responses-api/text-generation)** - Hasilkan respons teks berkualitas tinggi.
*   **[Manajemen Percakapan](/id/conversation-management)** - Dukungan asli untuk obrolan multi-turn stateful menggunakan `previous_response_id`.
*   **[Streaming](/id/sdk/responses-api/streaming)** - Pengiriman token waktu nyata melalui SSE.
*   **[Gambar & Visi](/id/sdk/responses-api/image-input)** - Proses gambar dan input multimodal.
*   **[Pemanggilan Tool](/id/sdk/responses-api/tool-calling)** - Integrasi mulus dengan fungsi eksternal.
*   **[Output Terstruktur](/id/capability/structured)** - Ekstraksi data JSON yang andal.

## Memulai

Berikut adalah contoh sederhana untuk menghasilkan respons teks menggunakan API `fetch`:

<CodeGroup>

```typescript quickstart.ts
const apiKey = process.env.NAI_API_KEY;
 
const response = await fetch('https://api.neosantara.xyz/v1/responses', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    Authorization: `Bearer ${apiKey}`,
  },
  body: JSON.stringify({
    model: 'grok-4.1-fast-non-reasoning',
    input: 'Apa ibu kota Indonesia?',
    store: true // Aktifkan manajemen status
  }),
});
 
const result = await response.json();
console.log(result.output_text);
```

```python quickstart.py
import os
import requests

api_key = os.getenv("NAI_API_KEY")

response = requests.post(
    "https://api.neosantara.xyz/v1/responses",
    headers={
        "Content-Type": "application/json",
        "Authorization": f"Bearer {api_key}"
    },
    json={
        "model": "grok-4.1-fast-non-reasoning",
        "input": "Apa ibu kota Indonesia?",
        "store": True
    }
)

result = response.json()
print(result["output_text"])
```

</CodeGroup>

## Parameter

### Parameter wajib

*   **`model`** (string): ID model Neosantara (misalnya, `grok-4.1-fast-non-reasoning`, `claude-3-haiku`).
*   **`input`** (string atau array): Input pengguna. Dapat berupa string sederhana atau array objek pesan.

### Parameter opsional

*   **`stream`** (boolean): Lakukan streaming respons menggunakan SSE. Default-nya `false`.
*   **`store`** (boolean): Apakah akan menyimpan status percakapan untuk turn mendatang.
*   **`previous_response_id`** (string): ID respons sebelumnya untuk melanjutkan percakapan stateful.
*   **`temperature`** (number): Mengontrol keacakan. Rentang: 0-2.
*   **`max_output_tokens`** (integer): Token maksimum yang akan dihasilkan dalam penyelesaian.
*   **`tools`** (array): Definisi tool untuk pemanggilan fungsi.
*   **`tool_choice`** (string): Mode pemilihan tool: `auto`, `required`, atau `none`.
*   **`reasoning`** (object): [Konfigurasi penalaran](/id/sdk/openai-compat/advanced#konfigurasi-reasoning) untuk model yang didukung.

## Penanganan kesalahan

API mengembalikan kode status HTTP standar dan respons kesalahan yang kompatibel dengan OpenAI.

### Kode kesalahan umum

*   `400 Bad Request` - Parameter permintaan tidak valid.
*   `401 Unauthorized` - Kunci API tidak valid atau hilang.
*   `404 Not Found` - Model atau endpoint tidak ditemukan.
*   `429 Too Many Requests` - Batas penggunaan atau saldo kredit terlampaui.
*   `500 Internal Server Error` - Kesalahan gateway atau penyedia.

### Format respons kesalahan

```json
{
  "error": {
    "message": "Permintaan tidak valid: parameter 'model' yang wajib diisi hilang",
    "type": "invalid_request_error",
    "param": "model",
    "code": "missing_parameter"
  }
}
```
