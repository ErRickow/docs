---
title: "Input File"
description: "Kirim dokumen, PDF, dan file lain bersama teks menggunakan API Responses."
---

[API Responses](/id/sdk/responses-api) mendukung input file selain gambar. Kamu dapat melampirkan PDF, dokumen teks, dan file lain melalui tiga metode: `file_id` yang sudah diunggah, `file_data` inline (base64), atau `file_url` publik.

---

## Unggah File

Unggah file kamu menggunakan endpoint `/v1/files` dengan `purpose=user_data`:

<CodeGroup>

```bash curl
curl https://api.neosantara.xyz/v1/files \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -F purpose="user_data" \
  -F file="@document.pdf"
```

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
import fs from 'fs';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1'
});

const file = await client.files.create({
  purpose: 'user_data',
  file: fs.createReadStream('document.pdf')
});
console.log(file.id); // file-abc123
```

```python Python (OpenAI SDK)
from openai import OpenAI
client = OpenAI(
    api_key=os.environ["NAI_API_KEY"],
    base_url="https://api.neosantara.xyz/v1"
)

file = client.files.create(
    purpose="user_data",
    file=open("document.pdf", "rb")
)
print(file.id)  # file-abc123
```

</CodeGroup>

## Menggunakan `input_file` dengan `file_id`

<CodeGroup>

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});

const response = await client.responses.create({
  model: 'claude-opus-4-6',
  input: [
    {
      role: 'user',
      content: [
        { type: 'input_file', file_id: 'file-abc123' },
        { type: 'input_text', text: 'Ringkas dokumen ini.' },
      ],
    },
  ],
});
console.log(response.output_text);
```

```python Python (OpenAI SDK)
from openai import OpenAI
client = OpenAI(
    api_key=os.environ["NAI_API_KEY"],
    base_url="https://api.neosantara.xyz/v1"
)

response = client.responses.create(
    model="claude-opus-4-6",
    input=[
        {
            "role": "user",
            "content": [
                {"type": "input_file", "file_id": "file-abc123"},
                {"type": "input_text", "text": "Ringkas dokumen ini."},
            ],
        }
    ],
)
print(response.output_text)
```

```bash curl
curl https://api.neosantara.xyz/v1/responses \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -d '{
    "model": "claude-opus-4-6",
    "input": [{
      "role": "user",
      "content": [
        { "type": "input_file", "file_id": "file-abc123" },
        { "type": "input_text", "text": "Ringkas dokumen ini." }
      ]
    }]
  }'
```

</CodeGroup>

## Menggunakan `input_file` dengan `file_data` inline

<CodeGroup>

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});

const fileContent = Buffer.from('Halo, ini adalah file tes.').toString('base64');

const response = await client.responses.create({
  model: 'claude-opus-4-6',
  input: [
    {
      role: 'user',
      content: [
        { type: 'input_file', file_data: fileContent, filename: 'halo.txt' },
        { type: 'input_text', text: 'Baca file ini.' },
      ],
    },
  ],
});
console.log(response.output_text);
```

```bash curl
curl https://api.neosantara.xyz/v1/responses \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -d '{
    "model": "claude-opus-4-6",
    "input": [{
      "role": "user",
      "content": [
        {
          "type": "input_file",
          "file_data": "SGVsbG8sIHRoaXMgaXMgYSB0ZXN0IGZpbGUu",
          "filename": "halo.txt"
        },
        { "type": "input_text", "text": "Baca file ini." }
      ]
    }]
  }'
```

</CodeGroup>

```bash curl

  -H "Content-Type: application/json" \
```

</CodeGroup>

<Note>
  Ukuran unggahan file dibatasi hingga 100 MB. Untuk input gambar melalui `input_image`, lihat [Input Gambar](/id/sdk/responses-api/image-input). Untuk Chat Completions, lihat [Input File](/id/sdk/openai-compat/file-input). Untuk endpoint kompatibel Anthropic, lihat [Lampiran File](/id/sdk/anthropic-compat/file-attachments).
</Note>
