Gunakan [API Responses](/id/sdk/responses-api) untuk menghasilkan respons teks berkualitas tinggi dari model AI. Tidak seperti API Chat Completions yang lama, API Responses memungkinkan Kamu untuk mengirimkan string sederhana atau objek pesan terstruktur sambil secara opsional mempertahankan status percakapan di server.

### Permintaan Dasar

Arahkan klien HTTP Kamu ke endpoint responses Neosantara.

<CodeGroup>

```typescript generate.ts
const apiKey = process.env.NAI_API_KEY;
 
const response = await fetch('https://api.neosantara.xyz/v1/responses', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    Authorization: `Bearer ${apiKey}`,
  },
  body: JSON.stringify({
    model: 'grok-4.1-fast-non-reasoning',
    input: [
      {
        type: 'message',
        role: 'user',
        content: 'Mengapa pengembang lebih menyukai mode gelap?',
      },
    ],
  }),
});
 
const result = await response.json();
console.log(result.output_text);
```

```python generate.py
import requests
import os

api_key = os.getenv("NAI_API_KEY")

response = requests.post(
    "https://api.neosantara.xyz/v1/responses",
    headers={
        "Content-Type": "application/json",
        "Authorization": f"Bearer {api_key}"
    },
    json={
        "model": "grok-4.1-fast-non-reasoning",
        "input": "Mengapa pengembang lebih menyukai mode gelap?"
    }
)

result = response.json()
print(result["output_text"])
```

</CodeGroup>

## Format Respons

Respons menyertakan teks yang dihasilkan dalam array `output`, bersama dengan informasi penggunaan token dan pembantu `output_text` yang nyaman untuk akses cepat.

```json
{
  "id": "resp_abc123",
  "object": "response",
  "model": "grok-4.1-fast-non-reasoning",
  "output_text": "Kebiasaan dan estetika memperkuat preferensi tersebut, tetapi ergonomi dan kontras adalah pendorong utamanya.",
  "output": [
    {
      "id": "msg_abc123",
      "type": "message",
      "role": "assistant",
      "content": [
        {
          "type": "output_text",
          "text": "Kebiasaan dan estetika memperkuat preferensi tersebut, tetapi ergonomi dan kontras adalah pendorong utamanya."
        }
      ]
    }
  ],
  "usage": {
    "prompt_tokens": 14,
    "completion_tokens": 18,
    "total_tokens": 32
  }
}
```

### Mengakses Output

*   **`output_text`**: String tingkat atas yang berisi respons teks lengkap yang digabungkan. Terbaik untuk aplikasi obrolan sederhana.
*   **`output`**: Array blok pesan terperinci. Gunakan ini jika Kamu perlu mengakses ID item tertentu, konten penalaran, atau panggilan tool.

Pelajari lebih lanjut tentang mengelola respons ini selama beberapa turn dalam panduan [Manajemen Percakapan](/id/conversation-management).
