# Generasi Gambar

Hasilkan gambar menggunakan model AI yang mendukung output multimodal. Fitur ini memungkinkan Kamu membuat gambar bersamaan dengan respons teks menggunakan model seperti **Gemini 2.0 Flash**.

Endpoint: `POST /v1/chat/completions`

### [Parameter Modalities](#parameter-modalities)

Sertakan parameter `modalities` untuk mengaktifkan generasi gambar:
*   **`modalities`**: `['text', 'image']` untuk teks dan gambar, atau `['image']` untuk gambar saja.

### [Contoh Permintaan](#contoh-permintaan)

```typescript
const completion = await openai.chat.completions.create({
  model: 'gemini-3-flash',
  messages: [{ role: 'user', content: 'Hasilkan pemandangan matahari terbenam.' }],
  // @ts-expect-error - ekstensi gateway
  modalities: ['text', 'image'],
});

<Note>
  **Ekstensi Gateway**: Parameter `modalities` serta bidang `message.images` dan `delta.images` yang dihasilkan adalah ekstensi gateway khusus Neosantara. Ini bukan bagian dari skema Chat Completions OpenAI standar dan mungkin tidak tersedia di endpoint kompatibel OpenAI lainnya.
</Note>
```
