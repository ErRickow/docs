---
title: "Input File"
description: "Kirim dokumen dan file bersama teks di Chat Completions menggunakan OpenAI SDK."
---

[API Chat Completions](/id/sdk/openai-compat/chat-completions) mendukung input file melalui OpenAI SDK. Kamu dapat melampirkan dokumen, gambar, dan file lain menggunakan `file.file_id` (unggah), `file.file_data` inline (base64), atau `file.file_url` (URL publik).

---

## Unggah File

Unggah file kamu menggunakan endpoint `/v1/files` dengan `purpose=user_data`:

<CodeGroup>

```bash curl
curl https://api.neosantara.xyz/v1/files \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -F purpose="user_data" \
  -F file="@document.pdf"
```

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
import fs from 'fs';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1'
});

const file = await client.files.create({
  purpose: 'user_data',
  file: fs.createReadStream('document.pdf')
});
console.log(file.id); // file-abc123
```

```python Python (OpenAI SDK)
from openai import OpenAI
client = OpenAI(
    api_key=os.environ["NAI_API_KEY"],
    base_url="https://api.neosantara.xyz/v1"
)

file = client.files.create(
    purpose="user_data",
    file=open("document.pdf", "rb")
)
print(file.id)  # file-abc123
```

</CodeGroup>

## Menggunakan `file.file_id` (Unggah)

<CodeGroup>

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});

const chat = await client.chat.completions.create({
  model: 'claude-opus-4-6',
  messages: [
    {
      role: 'user',
      content: [
        { type: 'file', file: { file_id: 'file-abc123' } },
        { type: 'text', text: 'Apa isi file ini?' },
      ],
    },
  ],
});
console.log(chat.choices[0].message.content);
```

```python Python (OpenAI SDK)
from openai import OpenAI
client = OpenAI(
    api_key=os.environ["NAI_API_KEY"],
    base_url="https://api.neosantara.xyz/v1"
)

chat = client.chat.completions.create(
    model="claude-opus-4-6",
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "file", "file": {"file_id": "file-abc123"}},
                {"type": "text", "text": "Apa isi file ini?"},
            ],
        }
    ],
)
print(chat.choices[0].message.content)
```

```bash curl
curl https://api.neosantara.xyz/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -d '{
    "model": "claude-opus-4-6",
    "messages": [{
      "role": "user",
      "content": [
        { "type": "file", "file": { "file_id": "file-abc123" } },
        { "type": "text", "text": "Apa isi file ini?" }
      ]
    }]
  }'
```

</CodeGroup>

## Menggunakan `file.file_data` (Base64 Inline)

<CodeGroup>

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});

const fileContent = Buffer.from('Halo, ini adalah file tes.').toString('base64');

const chat = await client.chat.completions.create({
  model: 'claude-opus-4-6',
  messages: [
    {
      role: 'user',
      content: [
        { type: 'file', file: { file_data: fileContent, filename: 'halo.txt' } },
        { type: 'text', text: 'Baca file ini.' },
      ],
    },
  ],
});
console.log(chat.choices[0].message.content);
```

```bash curl
curl https://api.neosantara.xyz/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -d '{
    "model": "claude-opus-4-6",
    "messages": [{
      "role": "user",
      "content": [
        { "type": "file", "file": { "file_data": "SGFsbywgaW5pIGFkYWxhaCBmaWxlIHRlcy4=", "filename": "halo.txt" } },
        { "type": "text", "text": "Baca file ini." }
      ]
    }]
  }'
```

</CodeGroup>

## Menggunakan `file.file_url` (URL Publik)

<CodeGroup>

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});

const chat = await client.chat.completions.create({
  model: 'claude-opus-4-6',
  messages: [
    {
      role: 'user',
      content: [
        { type: 'file', file_url: { url: 'https://example.com/document.pdf' } },
        { type: 'text', text: 'Ringkas dokumen ini.' },
      ],
    },
  ],
});
console.log(chat.choices[0].message.content);
```

```bash curl
curl https://api.neosantara.xyz/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -d '{
    "model": "claude-opus-4-6",
    "messages": [{
      "role": "user",
      "content": [
        { "type": "file", "file_url": { "url": "https://example.com/document.pdf" } },
        { "type": "text", "text": "Ringkas dokumen ini." }
      ]
    }]
  }'
```

</CodeGroup>

<Note>
  Ukuran unggahan file dibatasi hingga 100 MB. Untuk API Responses, lihat [Input File](/id/sdk/responses-api/file-input). Untuk endpoint kompatibel Anthropic, lihat [Lampiran File](/id/sdk/anthropic-compat/file-attachments).
</Note>
