## LangChain

*LangChain adalah framework untuk mengembangkan aplikasi yang ditenagai oleh large language models (LLMs).*

Instal paket `langchain-openai`.

```bash icon="terminal"
pip install langchain-openai
```

Buat instance klien Neosantara dan panggil model tersebut.

```python icon="python"
from langchain_openai import ChatOpenAI
import os

llm = ChatOpenAI(
    model="nusantara-base",
    api_key=os.environ.get("NAI_API_KEY"),
    base_url="https://api.neosantara.xyz/v1",
)

response = llm.invoke("Apa keuntungan menggunakan API AI regional?")
print(response.content)
```

Pelajari lebih lanjut di [Panduan LangChain](/id/langchain).

## Vercel AI SDK

*Vercel AI SDK adalah library Typescript yang kuat yang dirancang untuk membantu pengembang membangun aplikasi bertenaga AI.*

Instal Vercel AI SDK dan paket OpenAI Vercel.

```bash icon="terminal"
npm i ai @ai-sdk/openai
```

Buat instance klien Neosantara dan panggil fungsi generateText untuk menghasilkan teks.

```typescript
import { createOpenAICompatible } from '@ai-sdk/openai-compatible';
import { generateText } from 'ai';

const neosantara = createOpenAICompatible({
  name: 'neosantara_xyz',
  apiKey: process.env.NAI_API_KEY ?? "",
  baseURL: 'https://api.neosantara.xyz/v1',
});

const { text } = await generateText({
  model: neosantara('nusantara-base'),
  prompt: 'Tulis cerita tentang kucing yang menginginkan whiskas',
});

console.log(text)
```

Pelajari lebih lanjut di [Panduan Neosantara AI - Vercel AI SDK](/id/using-neosantara-with-vercels-ai-sdk).

## LlamaIndex

*LlamaIndex adalah framework data yang sederhana dan fleksibel untuk menghubungkan sumber data khusus ke model bahasa besar (LLM).*

Instal `llama-index`


Berikut adalah contoh kode untuk memulai dengan Llama Index + Neosantara AI:

```python icon="python"
from llama_index.llms.openai_like import OpenAILike

    model="nusantara-base",
    api_base="https://api.neosantara.xyz/v1",

    is_function_calling_model=True,
    temperature=0.1,

```

## CrewAI

*CrewAI adalah framework sumber terbuka untuk mengorkestrasi sistem agen AI.*

Instal `crewai`

```bash icon="terminal"
pip install crewai
export NAI_API_KEY=***
```

Bangun alur kerja multi-agen:

```python icon="python"
import os
from crewai import LLM, Task, Agent, Crew

llm = LLM(model="openai/nusantara-base",
          api_key=os.environ.get("NAI_API_KEY"),
          base_url="https://api.neosantara.xyz/v1"
        )

research_agent = Agent(
    llm = llm,
    role="Analis Riset",
    goal="Temukan dan ringkas informasi tentang topik tertentu",
    backstory="Kamu adalah peneliti berpengalaman dengan perhatian terhadap detail",
    verbose=True  # Aktifkan logging untuk debugging
)

research_task = Task(
    description="Lakukan riset menyeluruh tentang Agen AI.",
    expected_output="Daftar dengan 10 poin penting tentang Agen AI",
    agent=research_agent
)

# Eksekusi crew
crew = Crew(
    agents=[research_agent],
    tasks=[research_task],
    verbose=True
)

result = crew.kickoff()

# Mengakses output tugas
task_output = research_task.output

print(task_output)
```

Pelajari lebih lanjut di [Panduan CrewAI](/id/crewai) kami.

## PydanticAI

*PydanticAI adalah framework agen yang dibuat oleh tim Pydantic untuk menyederhanakan pembangunan alur kerja agen.*

Instal `pydantic-ai`

```bash icon="terminal"
pip install pydantic-ai
export NAI_API_KEY=***
```

Bangun agen PydanticAI menggunakan model Neosantara AI

```python icon="python"
from pydantic_ai import Agent
from pydantic_ai.models.openai import OpenAIChatModel
from pydantic_ai.providers.openai import OpenAIProvider
import os

# Hubungkan PydanticAI ke LLM di Neosantara
model = OpenAIChatModel(
    'nusantara-base',
    provider=OpenAIProvider(
        base_url="https://api.neosantara.xyz/v1",
        api_key=os.environ.get("NAI_API_KEY"),
    ),
)

# Siapkan agen
agent = Agent(
    model,
    system_prompt='Singkat, jawab dengan satu kalimat.',
)

result = agent.run_sync('Dari mana asal kata "hello world"?')  
print(result.output)
```

Pelajari lebih lanjut di [Panduan PydanticAI](/id/pydantic-ai) kami.

```python Python
# Set environment variables

# Otorisasi akses Gmail
auth_response = arcade_client.tools.authorize(
    tool_name="Google.SendEmail",
    user_id=USER_ID,
)

if auth_response.status != "completed":
    print(f"Klik tautan ini untuk mengotorisasi: {auth_response.url}")
    # Tunggu otorisasi selesai
```

## DSPy

*DSPy adalah framework yang memungkinkan kamu membangun sistem AI modular dengan kode, bukan prompting manual.*

Instal `dspy`

```bash icon="terminal"
pip install -U dspy
export NAI_API_KEY=***
```

Bangun agen penjawab pertanyaan

```python icon="python"
import dspy
import os

# Konfigurasi dspy dengan LLM dari Neosantara AI
lm = dspy.LM('openai/nusantara-base', 
             api_key=os.environ.get("NAI_API_KEY"), 
             api_base="https://api.neosantara.xyz/v1")

# Konfigurasi dspy untuk menggunakan LLM
dspy.configure(lm=lm)

# Memberikan agen akses ke Tools pencarian wikipedia
def search_wikipedia(query: str):
    results = dspy.ColBERTv2(url='https://id.m.wikipedia.org/wiki/Prabowo_Subianto')(query, k=1) # Ambil hasil teratas 1
    return results[0]['text'] if results else "Tidak ada informasi relevan ditemukan."


# siapkan modul ReAct dengan signature pertanyaan dan jawaban string
react = dspy.ReAct("question -> answer: str", tools=[search_wikipedia])

pred = react(question="kapan prabowo lahir?")

print(pred.answer)
```

Pelajari lebih lanjut di [Panduan DSPy](/id/dspy) kami.

## AutoGen(AG2)

*AG2 (sebelumnya AutoGen) adalah framework sumber terbuka untuk membangun dan mengorkestrasi agen AI.*

Instal `autogen`

```bash icon="terminal"
pip install autogen
export NAI_API_KEY=***
```

Bangun agen pengodean

```python icon="python"
import os
from pathlib import Path
from autogen import AssistantAgent, UserProxyAgent
from autogen.coding import LocalCommandLineCodeExecutor

config_list = [
    {
        # Mari pilih model Nusantara-Base
        "model": "nusantara-base",
        "base_url": "https://api.neosantara.xyz/v1",
        # Berikan kunci API Neosantara.xyz kamu di sini atau masukkan ke dalam variabel lingkungan NAI_API_KEY.
        "api_key": os.environ.get("NAI_API_KEY"),
        # Kami tentukan Tipe API sebagai 'openai' sehingga menggunakan kelas klien OpenAI
        "api_type": "openai",
        "stream": False,
        "price": [300/1000000*1000, 1500/1000000*1000] # Input: $300/Jt token, Output: $1500/Jt token
    }
]

# Menyiapkan eksekutor kode
workdir = Path("coding")
workdir.mkdir(exist_ok=True)
code_executor = LocalCommandLineCodeExecutor(work_dir=workdir)

# Menyiapkan agen

# UserProxyAgent akan mengeksekusi kode yang diberikan AssistantAgent
user_proxy_agent = UserProxyAgent(
    name="User",
    code_execution_config={"executor": code_executor},
    is_termination_msg=lambda msg: "FINISH" in msg.get("content"),
)

system_message = """Kamu adalah asisten AI yang membantu menulis kode dan pengguna mengeksekusinya.
Selesaikan tugas menggunakan keterampilan pengodean dan bahasa kamu.
"""

# AssistantAgent, menggunakan model nusantara-base Neosantara, akan mengambil permintaan pengodean dan mengembalikan kode
assistant_agent = AssistantAgent(
    name="NAI Coding Assistant",
    system_message=system_message,
    llm_config={"config_list": config_list},
)

# Mulai obrolan, dengan UserProxyAgent meminta pesan ke AssistantAgent
chat_result = user_proxy_agent.initiate_chat(
    assistant_agent,
    message="Berikan kode untuk menghitung jumlah bilangan prima dari 1 hingga 10000.",
)
```

Pelajari lebih lanjut di [Panduan Autogen](/id/autogen) kami.

```python icon="python"
from openai import OpenAI
import os

pc = Pinecone(
  api_key="PINECONE_API_KEY", 
  source_tag="NEOSANTARA_AI"
)
nai = OpenAI(
    api_key="NAI_API_KEY", 
    base_url="https://api.neosantara.xyz/v1"
)

# Buat indeks di pinecone
index = pc.create_index(
    name="serverless-index",
    dimension=1536,
    metric="cosine",
    spec=ServerlessSpec(cloud="aws", region="us-west-2"),
)

# Buat embedding di Neosantara AI
textToEmbed = "Tata surya kita mengorbit galaksi Bima Sakti dengan kecepatan sekitar 515.000 mph"
embeddings = nai.embeddings.create(
    model="nusa-embedding-0001", 
   input=textToEmbed
)
```
