Gunakan halaman ini untuk memilih jalur integrasi. Neosantara dapat dipakai melalui native framework provider saat tersedia, adapter OpenAI-compatible untuk dukungan ekosistem luas, dan API Anthropic-compatible untuk workflow bergaya Claude.

## Jalur yang direkomendasikan

<CardGroup cols={2}>
  <Card title="Agno native" icon="sparkles" href="/id/agno">
    Gunakan `agno.models.neosantara.Neosantara` untuk agent multimodal, tool use, streaming, dan switching model.
  </Card>

  <Card title="Any-LLM native" icon="blocks" href="/id/any-llm">
    Gunakan provider id `neosantara` dengan `NEOSANTARA_API_KEY` saat membutuhkan satu interface Python lintas provider.
  </Card>
</CardGroup>

## Adapter OpenAI-compatible

Framework berikut dapat memakai Neosantara dengan mengarahkan client OpenAI-compatible ke `https://api.neosantara.xyz/v1`.

<CardGroup cols={3}>
  <Card title="LangChain" icon="link" href="/id/langchain">
    Gunakan `langchain-openai` dengan base URL Neosantara.
  </Card>

  <Card title="CrewAI" icon="users" href="/id/crewai">
    Jalankan agent CrewAI melalui model string Neosantara yang OpenAI-compatible.
  </Card>

  <Card title="DSPy" icon="route" href="/id/dspy">
    Konfigurasikan DSPy dengan `openai/<model-id>` dan API base Neosantara.
  </Card>

  <Card title="Pydantic AI" icon="badge-check" href="/id/pydantic-ai">
    Gunakan Pydantic AI dengan Neosantara untuk workflow agent typed dan structured output.
  </Card>

  <Card title="LlamaIndex" icon="database" href="/id/llama-index">
    Hubungkan workflow RAG LlamaIndex ke model chat dan embedding Neosantara.
  </Card>

  <Card title="AutoGen / AG2" icon="bot" href="/id/autogen">
    Konfigurasikan agent AutoGen dengan setting OpenAI-compatible Neosantara.
  </Card>
</CardGroup>
