Model Neosantara mampu berinteraksi dengan tools (juga dikenal sebagai fungsi), memungkinkan Kamu memperluas kemampuan AI untuk melakukan berbagai tugas yang lebih luas, seperti mengambil data waktu nyata, melakukan perhitungan, atau berinteraksi dengan sistem eksternal.

Berikut adalah contoh cara memberikan tools ke Neosantara menggunakan chat completions API:

<CodeGroup>
  ```bash Shell
  curl https://api.neosantara.xyz/v1/chat/completions \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer $NEOSANTARA_API_KEY" \
    -d '{
      "model": "nusantara-base",
      "max_tokens": 1024,
      "tools": [
        {
          "type": "function",
          "function": {
            "name": "dapatkan_cuaca",
            "description": "Dapatkan cuaca saat ini di lokasi tertentu",
            "parameters": {
              "type": "object",
              "properties": {
                "lokasi": {
                  "type": "string",
                  "description": "Kota dan negara, misal Jakarta, ID"
                }
              },
              "required": ["lokasi"]
            }
          }
        }
      ],
      "messages": [
        {
          "role": "user",
          "content": "Bagaimana cuaca di Jakarta?"
        }
      ]
    }'
  ```

  ```python Python
  from openai import OpenAI

  client = OpenAI(
      api_key="KUNCI_API_NEOSANTARA_KAMU",
      base_url="https://api.neosantara.xyz/v1"
  )

  response = client.chat.completions.create(
      model="nusantara-base",
      max_tokens=1024,
      tools=[
          {
              "type": "function",
              "function": {
                  "name": "dapatkan_cuaca",
                  "description": "Dapatkan cuaca saat ini di lokasi tertentu",
                  "parameters": {
                      "type": "object",
                      "properties": {
                          "lokasi": {
                              "type": "string",
                              "description": "Kota dan negara, misal Jakarta, ID",
                          }
                      },
                      "required": ["lokasi"],
                  },
              }
          }
      ],
      messages=[{"role": "user", "content": "Bagaimana cuaca di Jakarta?"}],
  )
  print(response)
  ```

</CodeGroup>

## Bagaimana Penggunaan Tools Bekerja

Fungsionalitas penggunaan tools Neosantara mengikuti pola umum, mirip dengan pemanggilan fungsi OpenAI, di mana Kamu memberikan deskripsi tools yang tersedia kepada model, dan model memutuskan apakah dan bagaimana menggunakannya.

Integrasikan tools sisi klien dengan Neosantara dalam langkah-langkah ini:

<Steps>
  <Step title="Berikan Neosantara tools dan prompt pengguna">
    * Definisikan tools dengan nama, deskripsi, dan skema input (parameter) dalam permintaan API Kamu di dalam parameter `tools`.
    * Sertakan prompt pengguna yang mungkin memerlukan tools ini, misal, "Bagaimana cuaca di Jakarta?"
  </Step>

  <Step title="Neosantara memutuskan untuk menggunakan tools">
    * Model menilai apakah ada tools yang dapat membantu pertanyaan pengguna.
    * Jika ya, model membangun objek `tool_calls` dalam responsnya, berisi nama tools yang akan dipanggil dan argumen (input) untuk tools tersebut.
    * Respons API akan memiliki `finish_reason` berupa `tool_calls`.
  </Step>

  <Step title="Eksekusi tools dan kembalikan hasil">
    * Aplikasi Kamu mengekstrak nama tools dan input dari respons `tool_calls` Neosantara.
    * Aplikasi Kamu kemudian mengeksekusi kode tools yang sebenarnya di sistem Kamu.
    * Kembalikan hasilnya ke model dalam pesan `user` baru dengan peran `tool`, berisi `tool_call_id` dan `content` dari output tools.
  </Step>

  <Step title="Neosantara menggunakan hasil tools untuk merumuskan respons">
    * Neosantara menganalisis hasil tools yang Kamu berikan untuk membuat respons bahasa alami finalnya terhadap prompt pengguna asli.
  </Step>
</Steps>

Catatan: Langkah 3 dan 4 bersifat opsional. Untuk beberapa alur kerja, permintaan penggunaan tools Neosantara (langkah 2) mungkin adalah semua yang Kamu butuhkan.

***

## Harga

Permintaan penggunaan tools diberi harga berdasarkan:

1. Total jumlah token input yang dikirim ke model (termasuk token dari definisi tools).
2. Jumlah token output yang dihasilkan (termasuk panggilan tools yang dihasilkan oleh model).

Lihat [dokumentasi harga](/id/about/token-credits) kami untuk harga per model dan tingkatan penggunaan saat ini.
