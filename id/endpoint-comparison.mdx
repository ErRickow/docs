## Rekomendasi Kasus Penggunaan

### Pilih Kompatibel OpenAI Jika

- âœ… **Memigrasikan aplikasi OpenAI yang ada**
- âœ… **Menggunakan library seperti LangChain, LlamaIndex**
- âœ… **Membutuhkan kompatibilitas SDK maksimum**
- âœ… **Tim sudah memahami format OpenAI**
- âœ… **Prioritas integrasi cepat**

**Contoh Kasus Penggunaan:**

```python
# Sempurna untuk aplikasi OpenAI yang sudah ada
from openai import OpenAI

client = OpenAI(base_url="https://api.neosantara.xyz/v1")
response = client.chat.completions.create(
    model="nusantara-base",
    messages=[{"role": "user", "content": "Halo!"}]
)
```

### Pilih Kompatibel Anthropic Jika

- âœ… **Menggunakan Claude Code atau SDK Anthropic**
- âœ… **Membutuhkan kemampuan pemanggilan tools tingkat lanjut**
- âœ… **Lebih menyukai format pesan Anthropic**
- âœ… **Membangun dengan tools ekosistem Claude**
- âœ… **Memerlukan mode penalaran yang canggih**

**Contoh Kasus Penggunaan:**

```python
# Terbaik untuk ekosistem Anthropic
import anthropic

client = anthropic.Anthropic(base_url="https://api.neosantara.xyz/anthropic")
message = client.messages.create(
    model="nusantara-base",
    messages=[{"role": "user", "content": "Halo!"}]
)
```

### Pilih API Responses Jika

- âœ… **Memulai aplikasi baru**
- âœ… **Menginginkan percakapan yang stateful**
- âœ… **Membutuhkan input/output yang disederhanakan**
- âœ… **Membangun aplikasi berbasis agen**
- âœ… **Lebih menyukai desain API modern**

**Contoh Kasus Penggunaan:**

```python
# Ideal untuk aplikasi baru
from openai import OpenAI

client = OpenAI(base_url="https://api.neosantara.xyz/v1")
response = client.responses.create(
    model="nusantara-base",
    input="Halo! Ceritakan tentang dirimu.",
    store=True  # Manajemen percakapan otomatis
)
```

## Panduan Migrasi

### Dari Kompatibel OpenAI ke API Responses

```python
# Sebelum (Kompatibel OpenAI)
response = client.chat.completions.create(
    model="nusantara-base",
    messages=[
        {"role": "system", "content": "Kamu membantu."},
        {"role": "user", "content": "Halo!"}
    ],
    tools=[...],
    stream=True
)

# Sesudah (API Responses)
response = client.responses.create(
    model="nusantara-base",
    input="Halo! Ceritakan tentang dirimu.",
    system="Kamu membantu.",
    tools=[...],
    store=True,  # Manajemen status otomatis
    stream=True
)

# Akses output yang disederhanakan
print(response.output_text)
```

### Dari Kompatibel Anthropic ke API Responses

```python
# Sebelum (Kompatibel Anthropic)
message = client.messages.create(
    model="nusantara-base",
    system="Kamu membantu.",
    messages=[
        {"role": "user", "content": [{"type": "text", "text": "Halo!"}]}
    ],
    tools=[...]
)

# Sesudah (API Responses)
response = client.responses.create(
    model="nusantara-base",
    input="Halo! Ceritakan tentang dirimu.",
    system="Kamu membantu.",
    tools=[...],
    store=True
)

# Akses konten secara langsung
print(response.output_text)
```

## Pertimbangan Performa

### Perbandingan Latensi

| Endpoint | Latensi Rata-rata | Latensi P95 | Catatan |

## Contoh Kode Berdasarkan Kasus Penggunaan

### Aplikasi Chat

<CodeGroup>

```python
from openai import OpenAI

client = OpenAI(base_url="https://api.neosantara.xyz/v1")

def chat_with_history(message_history, new_message):
    response = client.chat.completions.create(
        model="nusantara-base",
        messages=message_history + [new_message]
    )
    return response.choices[0].message.content
```

```python
import anthropic

client = anthropic.Anthropic(base_url="https://api.neosantara.xyz/anthropic")

def chat_with_history(message_history, new_message):
    response = client.messages.create(
        model="nusantara-base",
        messages=message_history + [new_message]
    )
    return response.content[0].text
```

```python
from openai import OpenAI

client = OpenAI(base_url="https://api.neosantara.xyz/v1")

def chat_stateful(user_input, previous_id=None):
    response = client.responses.create(
        model="nusantara-base",
        input=user_input,
        previous_response_id=previous_id,
        store=True
    )
    return response.output_text, response.id
```

## Praktik Terbaik

### Rekomendasi Umum

1. **Gunakan variabel lingkungan** untuk kunci API
2. **Terapkan penanganan kesalahan yang tepat** untuk semua endpoint
3. **Gunakan streaming** untuk aplikasi real-time
4. **Atur timeout yang sesuai** untuk kasus penggunaan Kamu
5. **Pantau penggunaan token** di semua endpoint

### Tip Spesifik Endpoint

**Kompatibel OpenAI:**

- Kelola riwayat pesan di sisi klien
- Gunakan format pemanggilan fungsi standar
- Manfaatkan library OpenAI yang ada

**Kompatibel Anthropic:**

- Gunakan format skema tools
- Manfaatkan mode berpikir untuk tugas yang kompleks
- Gunakan blok konten untuk input multimodal

**API Responses:**

- Aktifkan `store` untuk manajemen status
- Gunakan `previous_response_id` untuk kelanjutan percakapan
- Manfaatkan format output terpadu
