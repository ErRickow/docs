## Ikhtisar

Panduan ini menjelaskan cara menggunakan **Neosantara** dan **SDK E2B Code Interpreter** untuk membuat sandbox cloud yang aman yang didukung oleh [Firecracker](https://github.com/firecracker-microvm/firecracker). Sandbox ini menyertakan server Jupyter yang sedang berjalan, yang memungkinkan model bahasa besar (LLM) untuk menjalankan kode Python untuk tugas-tugas seperti analisis dan visualisasi data.

---

## Prasyarat

Untuk memulai, pastikan Kamu memiliki hal-hal berikut:

<Info>
  Daftar di [Neosantara](https://app.neosantara.xyz/api-keys) untuk menerima **kredit awal gratis**. Tidak diperlukan kartu kredit.
</Info>

<Card icon="key" href="https://e2b.dev/docs" title="Dapatkan Kunci API E2B Kamu" horizontal>
  Dapatkan kunci API dari [E2B](https://e2b.dev/docs) untuk menggunakan Code Interpreter.
</Card>

- **Node.js**: Versi 16 atau lebih tinggi.
- **Python**: Versi 3.6 atau lebih tinggi.
- **Paket Python yang Diperlukan**:
  - [`openai`](https://pypi.org/project/openai/): Untuk berinteraksi dengan API Neosantara.
  - [`e2b_code_interpreter`](https://pypi.org/project/e2b-code-interpreter/): Untuk menjalankan kode di sandbox E2B.
  - [`python-dotenv`](https://pypi.org/project/python-dotenv/): Untuk mengelola variabel lingkungan.

## Instalasi

Instal paket Python yang diperlukan menggunakan pip:

```bash icon="terminal"
pip install e2b_code_interpreter==1.0.0 python-dotenv openai -q
```

Untuk detail tambahan tentang metode yang tersedia, lihat [dokumentasi E2B](https://e2b.dev/docs/quickstart).

## Contoh Kode

Di bawah ini adalah contoh lengkap yang menunjukkan cara menyiapkan Code Interpreter, mengunggah kumpulan data, dan membuat visualisasi (misalnya, grafik regresi linier PDB per kapita vs. angka harapan hidup).

```python icon="python"
import os
from dotenv import load_dotenv
from openai import OpenAI
from e2b_code_interpreter import Sandbox
import re
import json

# Muat variabel lingkungan
load_dotenv()

# Kunci API
NAI_API_KEY = os.getenv("NAI_API_KEY")
E2B_API_KEY = os.getenv("E2B_API_KEY")

# Pemilihan model
MODEL_NAME = 'nusantara-base'  # Alternatif: 'archipelago-70b', 'Llama-3.3-Nemotron-Super-49B'

# Prompt sistem untuk LLM
SYSTEM_PROMPT = """
Kamu adalah seorang ilmuwan data Python. Kamu diberikan tugas untuk diselesaikan dan Kamu menjalankan kode Python untuk menyelesaikannya.

Informasi tentang kumpulan data csv:
- Terletak di file `/home/user/data.csv`
- File CSV menggunakan , sebagai pemisah
- Memiliki kolom-kolom berikut (termasuk contoh):
    - country: "Argentina", "Australia"
    - Region: "SouthAmerica", "Oceania"
    - Surface area (km2): misalnya, 2780400
    - Population in thousands (2017): misalnya, 44271
    - Population density (per km2, 2017): misalnya, 16.2
    - Sex ratio (m per 100 f, 2017): misalnya, 95.9
    - GDP: Gross domestic product (million current US$): misalnya, 632343
    - GDP growth rate (annual %, const. 2005 prices): misalnya, 2.4
    - GDP per capita (current US$): misalnya, 14564.5
    - Economy: Agriculture (% of GVA): misalnya, 10.0
    - Economy: Industry (% of GVA): misalnya, 28.1
    - Economy: Services and other activity (% of GVA): misalnya, 61.9
    - Employment: Agriculture (% of employed): misalnya, 4.8
    - Employment: Industry (% of employed): misalnya, 20.6
    - Employment: Services (% of employed): misalnya, 74.7
    - Unemployment (% of labour force): misalnya, 8.5
    - Employment: Female (% of employed): misalnya, 43.7
    - Employment: Male (% of employed): misalnya, 56.3
    - Labour force participation (female %): misalnya, 48.5
    - Labour force participation (male %): misalnya, 71.1
    - International trade: Imports (million US$): misalnya, 59253
    - International trade: Exports (million US$): misalnya, 57802
    - International trade: Balance (million US$): misalnya, -1451
    - Education: Government expenditure (% of GDP): misalnya, 5.3
    - Health: Total expenditure (% of GDP): misalnya, 8.1
    - Health: Government expenditure (% of total health expenditure): misalnya, 69.2
    - Health: Private expenditure (% of total health expenditure): misalnya, 30.8
    - Health: Out-of-pocket expenditure (% of total health expenditure): misalnya, 20.2
    - Health: External health expenditure (% of total health expenditure): misalnya, 0.2
    - Education: Primary gross enrollment ratio (f/m per 100 pop): misalnya, 111.5/107.6
    - Education: Secondary gross enrollment ratio (f/m per 100 pop): misalnya, 104.7/98.9
    - Education: Tertiary gross enrollment ratio (f/m per 100 pop): misalnya, 90.5/72.3
    - Education: Mean years of schooling (female): misalnya, 10.4
    - Education: Mean years of schooling (male): misalnya, 9.7
    - Urban population (% of total population): misalnya, 91.7
    - Population growth rate (annual %): misalnya, 0.9
    - Fertility rate (births per woman): misalnya, 2.3
    - Infant mortality rate (per 1,000 live births): misalnya, 8.9
    - Life expectancy at birth, female (years): misalnya, 79.7
    - Life expectancy at birth, male (years): misalnya, 72.9
    - Life expectancy at birth, total (years): misalnya, 76.4
    - Military expenditure (% of GDP): misalnya, 0.9
    - Population, female: misalnya, 22572521
    - Population, male: misalnya, 21472290
    - Tax revenue (% of GDP): misalnya, 11.0
    - Taxes on income, profits and capital gains (% of revenue): misalnya, 12.9
    - Urban population (% of total population): misalnya, 91.7

Secara umum, Kamu mengikuti aturan-aturan ini:
- SELALU FORMAT RESPONS KAMU DALAM MARKDOWN
- SELALU MERESPONS HANYA DENGAN KODE DALAM BLOK KODE SEPERTI INI:
`\
```python
{kode}
```
`\
- Kode Python berjalan di buku catatan (notebook) Jupyter.
- Setiap blok kode Python dijalankan dalam sel terpisah.
- Tampilkan visualisasi menggunakan matplotlib atau pustaka lain secara langsung di buku catatan.
- Kamu memiliki akses ke internet dan dapat membuat permintaan API.
- Kamu dapat membaca/menulis file di sistem file sandbox.
- Instal paket pip apa pun menggunakan `!pip install {paket}` jika diperlukan (paket analisis data umum sudah terinstal sebelumnya).
- Semua kode berjalan di lingkungan sandbox yang aman.
"""

# Fungsi untuk mengeksekusi kode di sandbox
def code_interpret(e2b_code_interpreter, code):
    print("Menjalankan code interpreter...")
    exec = e2b_code_interpreter.run_code(
        code,
        on_stderr=lambda stderr: print("[Code Interpreter] Kesalahan:", stderr),
        on_stdout=lambda stdout: print("[Code Interpreter] Output:", stdout),
    )
    if exec.error:
        print("[KESALAHAN Code Interpreter]", exec.error)
        return None
    return exec.results

# Fungsi untuk mengekstrak kode Python dari respons LLM
def match_code_blocks(llm_response):
    pattern = re.compile(r"```python\s*([\s\S]*?)\s*```")
    match = pattern.search(llm_response)
    return match.group(1) if match else ""

# Fungsi untuk berinteraksi dengan LLM dan mengeksekusi kode
def chat_with_llm(e2b_code_interpreter, user_message):
    print(f"\n{'='*50}\nPesan pengguna: {user_message}\n{'='*50}")
    messages = [
        {"role": "system", "content": SYSTEM_PROMPT},
        {"role": "user", "content": user_message},
    ]
    client = OpenAI(api_key=NAI_API_KEY, base_url="https://api.neosantara.xyz/v1")
    response = client.chat.completions.create(model=MODEL_NAME, messages=messages)
    response_message = response.choices[0].message
    python_code = match_code_blocks(response_message.content)
    if python_code:
        return code_interpret(e2b_code_interpreter, python_code)
    else:
        print(f"Tidak ada kode Python yang ditemukan dalam respons: {response_message}")
        return []

# Fungsi untuk mengunggah kumpulan data ke sandbox
def upload_dataset(code_interpreter):
    print("Mengunggah kumpulan data ke sandbox Code Interpreter...")
    dataset_path = "./data.csv"
    if not os.path.exists(dataset_path):
        raise FileNotFoundError("File kumpulan data tidak ditemukan di ./data.csv")
    try:
        with open(dataset_path, "rb") as f:
            file_buffer = f.read()
            remote_path = code_interpreter.files.write('data.csv', file_buffer)
        if not remote_path:
            raise ValueError("Gagal mengunggah kumpulan data")
        print("Kumpulan data diunggah ke:", remote_path)
        return remote_path
    except Exception as error:
        print("Kesalahan selama pengunggahan file:", error)
        raise error

# Eksekusi utama
with Sandbox(api_key=E2B_API_KEY) as code_interpreter:
    # Unggah kumpulan data
    upload_dataset(code_interpreter)
    
    # Contoh tugas: Buat grafik regresi linier
    code_results = chat_with_llm(
        code_interpreter,
        "Buat grafik yang menunjukkan regresi linier PDB per kapita vs. angka harapan hidup dari kumpulan data. Saring nilai yang hilang atau tidak valid.",
    )
    
    if code_results:
        first_result = code_results[0]
        print("Visualisasi berhasil dibuat!")
    else:
        raise Exception("Tidak ada hasil dari code interpreter")

# Tampilkan visualisasi (misalnya, output PNG)
first_result

```

## Pemecahan Masalah

- **Masalah Kunci API**: Pastikan kunci API Neosantara dan E2B Kamu valid dan disetel dengan benar di file `.env`.
