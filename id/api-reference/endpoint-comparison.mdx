## Contoh Penggunaan

### Chat Aplikasi

```python
# Cocok untuk aplikasi OpenAI yang sudah ada
from openai import OpenAI
client = OpenAI(base_url="https://api.neosantara.xyz/v1")
response = client.chat.completions.create(model="nusantara-base", messages=[{"role":"user","content":"Halo!"}])
```

### Anthropic Compatible

```python
import anthropic
client = anthropic.Anthropic(base_url="https://api.neosantara.xyz/anthropic")
message = client.messages.create(model="nusantara-base", messages=[{"role":"user","content":"Halo!"}])
```

### Responses API

```python
from openai import OpenAI
client = OpenAI(base_url="https://api.neosantara.xyz/v1")
response = client.responses.create(model="nusantara-base", input="Halo!")
```

## Migrasi

### Dari OpenAI Compatible ke Responses API

```python
# OpenAI Compatible contoh
response = client.chat.completions.create(model="nusantara-base", messages=[{"role":"system","content":"Anda membantu."}, {"role":"user","content":"Halo!"}], stream=True)

response = client.responses.create(model="nusantara-base", input="Halo! Ceritakan tentang dirimu.", system="Anda membantu.", store=True, stream=True)
print(response.output_text)
```
