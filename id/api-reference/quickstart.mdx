## Siapkan Akun

Langkah awal, buat akun Neosantara di sini:

<Card icon="key" href="https://app.neosantara.xyz/signup" title="Buat Akun Gratis" horizontal>
  Dapatkan 7.500 Token gratis setiap bulan. **Tanpa perlu kartu kredit.**

Setelah daftar, kunjungi halaman "Kunci API" dan klik "Buat Kunci Baru". Salin kunci API-nya, simpan dengan aman, dan jangan dibagikan ke siapa pun.

### Panggilan API Pertama

API Neosantara kompatibel dengan format OpenAI. Dengan sedikit pengaturan, kamu bisa pakai OpenAI SDK atau perangkat lunak yang mendukung API OpenAI untuk akses Neosantara.

<Note>
**Neosantara AI** belum punya SDK sendiri. 
Gunakan fetch atau akses melalui OpenAI untuk saat ini.
</Note>

| PARAM | NILAI                           |
| -------- | ------------------------------------- |
| baseURL     | `https://api.neosantara.xyz/v1`                     |
| apiKey      | `<Kunci API Neosantara Anda>`                          |

### Panggil API Chat

Berikut contoh tanpa streaming. Jika ingin respons streaming, cukup set parameter `stream` ke `true`.

<CodeGroup>

```javascript openNAI.js lines icon="square-js"
// Install OpenAI SDK terlebih dahulu: `npm install openai`

const OpenAI = require("openai"); // atau gunakan import

const Nai = new OpenAI({
        baseURL: 'https://api.neosantara.xyz/v1',
        apiKey: '<Kunci API NEOSANTARA>'
});

async function main() {
  const completion = await Nai.chat.completions.create({
    messages: [{ role: "user", content: "Apa itu Neosantara AI?" }],
    model: "nusantara-base",
  });

  console.log(completion.choices[0].message.content);
}

main();
```

```python openNAI.py lines icon="python"
# Install OpenAI SDK terlebih dahulu: `pip3 install openai`

from openai import OpenAI

client = OpenAI(api_key="<Kunci API NEOSANTARA>", base_url="https://api.neosantara.xyz")

response = client.chat.completions.create(
    model="nusantara-base",
    messages=[
        {"role": "system", "content": "Anda adalah asisten yang siap membantu."},
        {"role": "user", "content": "Halo!"}
    ],
    stream=False
)

print(response.choices[0].message.content)
```

```bash curlNAI.sh lines icon="terminal"
curl https://api.neosantara.xyz/v1/chat/completions \

        ],
        "stream": false
      }'
```
