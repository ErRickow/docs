## URL Dasar

URL dasar untuk semua permintaan ke API Neosantara adalah:

```plaintext with prefix /v1
https://api.neosantara.xyz/v1
```

## Otentikasi

Semua endpoint Neosantara menggunakan kunci API untuk otentikasi.
[Dapatkan kunci API gratis Anda](https://app.neosantara.xyz/api-keys).

```bash icon="terminal"
curl -X POST https://api.neosantara.xyz/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer KUNCI_API_NEOSANTARA_ANDA" \
  -d '{"query": "Apakah 2045 masa depan Indonesia?"}'
```
