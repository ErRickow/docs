Neosantara API mendukung beberapa metode autentikasi pada endpoint-endpointnya.

## Metode
- Bearer Token (API Key di header Authorization)
- API Key header (misal `x-api-key`)
- Variabel lingkungan untuk pengembangan lokal (rekomendasi)

## Cara Penggunaan
- Atur header Authorization: Bearer <API_KEY>
- Atau berikan API key melalui header seperti `x-api-key` sesuai dengan library klien.
- Untuk aplikasi berbasis browser, gunakan backend yang aman untuk memproxy permintaan agar kunci tetap rahasia.

## Contoh

- Python
```
import requests
headers = {"Authorization": "Bearer <YOUR_API_KEY>"}
payload = {"model": "nusantara-base", "messages": [{"role": "user", "content": "Halo"}]}
resp = requests.post("https://api.neosantara.xyz/v1/chat/completions", json=payload, headers=headers)
```

- JavaScript
```
const res = await fetch("https://api.neosantara.xyz/v1/chat/completions", {
  method: 'POST',
  headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer <YOUR_API_KEY>' },
  body: JSON.stringify(payload)
})
```
