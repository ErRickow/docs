title: 'Kecocokan Anthropic'
description: 'Gunakan Anthropic SDK dan format API dengan Antarmuka Neosantara AI.'
---

Neosantara AI menyediakan dukungan penuh terhadap format API Anthropic Messages, memungkinkan Anda menggunakan SDK Anthropic asli dan alat-alatnya dengan model Neosantara. Ini mempermudah integrasi ke dalam alur kerja Anthropic yang sudah ada.

## Mulai Cepat

Dapatkan pengalaman terbaik dengan menggunakan Anthropic Messages API secara native.

<Steps>
  <Step title="Pasang Anthropic SDK">
    ```bash
    pip install anthropic
    ```
  </Step>

  <Step title="Konfigurasi Klien">
    ```bash
    export ANTHROPIC_BASE_URL=https://api.neosantara.xyz/anthropic
    export ANTHROPIC_API_KEY=<API_KEY_ANTHROPIC_ANDA>
    ```
  </Step>
  
  <Step title="Memanggil API">
    ```python
    import anthropic
    client = anthropic.Anthropic(base_url="https://api.neosantara.xyz/anthropic", api_key="<YOUR_API_KEY>")
    message = client.messages.create(
        model="nusantara-base",
        messages=[{"role": "user", "content": "Halo!"}]
    )
    print(message.content[0].text)
    ```
  </Step>
</Steps>

## Pemanggilan API Anthropic

<Steps>
  <Step title="Install Anthropic SDK">
    ```bash
    pip install anthropic
    ```
  </Step>
  <Step title="Konfigurasi Lingkungan">
    ```bash
    export ANTHROPIC_BASE_URL=https://api.neosantara.xyz/anthropic
    export ANTHROPIC_API_KEY=${NAI_API_KEY}
    ```
  </Step>
  <Step title="Contoh Pemanggilan API">
    ```python
    import anthropic
    client = anthropic.Anthropic(base_url="https://api.neosantara.xyz/anthropic")
    message = client.messages.create(
        model="nusantara-base",
        messages=[{"role": "user", "content": "Halo!"}]
    )
    print(message.content[0].text)
    ```
  </Step>
</Steps>

## Tabel Kompatibilitas

| Parameter | Kebutuhan | Tipe | Claude SDK | Default | Catatan |
|-----------|-----------|------|------------|---------|---------|
| model | Ya | string | Ya | - | Gunakan ID model Neosantara |
| messages | Ya | array | Ya | - | Struktur pesan kontinyu |
| max_tokens | Ya | integer | Ya | - | Token maksimal pengeluaran |

## Perbandingan Optimal
- Gunakan Anthropic Compatible jika Anda bekerja dalam ekosistem Anthropic dan memanfaatkan alat tooling khusus Anthropic.
- Gunakan OpenAI Compatible jika Anda memiliki ekosistem OpenAI yang luas dan sudah terbiasa dengan format OpenAI.

## Migrasi

- Jika sebelumnya Anda menggunakan OpenAI compatibility, lihat panduan migrasi untuk memetakan format pesan, sistem prompt, dan tool calling ke format Anthropic.
