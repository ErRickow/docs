API Neosantara menggunakan kode status HTTP standar. Halaman ini mendokumentasikan kesalahan umum, artinya, dan langkah-langkah remediasi yang disarankan.

## Kesalahan Umum

- 400 `invalid_request_error` — Permintaan tidak terbentuk dengan benar atau parameter yang diperlukan hilang.
- 401 `authentication_error` — Kunci API / token hilang atau tidak valid.
- 403 `permission_error` — Akses ke sumber daya dilarang.
- 429 `rate_limit_error` — Kamu telah melampaui batas kecepatan Kamu.
- 500 `api_error` — Kesalahan server internal.
- 503 `overloaded_error` — Layanan untuk sementara tidak tersedia.

## Contoh Respons Kesalahan

```json icon="code"
{
  "error": {
    "code": "invalid_request_error",
    "message": "Parameter 'messages' wajib diisi.",
    "type": "invalid_request_error"
  }
}
```

## Upaya Ulang & Pemulihan

- Gunakan exponential backoff untuk upaya ulang.
- Terapkan pola circuit breaker untuk menghindari lonjakan upaya ulang.
- Validasi input sebelum mencoba ulang.

## Contoh Kode (Language Bindings)

- Python

```python icon="python"
import requests
headers = {"Authorization": "Bearer KUNCI_API_NEOSANTARA_KAMU"}
payload = {"model": "nusantara-base", "messages": [{"role": "user", "content": "Halo"}]}
resp = requests.post("https://api.neosantara.xyz/v1/chat/completions", json=payload, headers=headers)
```

- JavaScript

```js
const payload = {
  model: "nusantara-base",
  messages: [{ role: "user", content: "Halo" }]
};

const res = await fetch("https://api.neosantara.xyz/v1/chat/completions", {
  method: 'POST',
  headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer KUNCI_API_NEOSANTARA_KAMU' },
  body: JSON.stringify(payload)
})
```
