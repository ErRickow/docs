## Langkah Pertama

Neosantara bertindak sebagai **gateway terpadu**, yang memungkinkan kamu mengakses berbagai model mutakhir dari penyedia berbeda menggunakan satu kunci API dan SDK pilihan kamu.

<Steps>
  <Step title="Dapatkan Kunci API kamu">
    1.  Daftar atau Masuk di [Aplikasi Neosantara](https://app.neosantara.xyz).
    2.  Buka [Dashboard](https://app.neosantara.xyz/dashboard) untuk melihat kredit gratis awal kamu.
    3.  Buat kunci baru di bagian [API Keys](https://app.neosantara.xyz/api-keys).
    4.  Salin kunci kamu (dimulai dengan `nsk_`).

    <CardGroup cols={2}>
      <Card title="Dashboard" icon="link" href="https://app.neosantara.xyz/dashboard">
        Kelola akun, tagihan, dan statistik.
      </Card>
      <Card title="API Keys" icon="key" href="https://app.neosantara.xyz/api-keys">
        Kelola kredensial akses kamu.
      </Card>
    </CardGroup>
  </Step>

  <Step title="Pilih Model kamu">
    Akses model terbaik dunia melalui satu antarmuka. Ganti model hanya dengan mengubah string `model`.

    *   **Unggulan Penalaran**: `deepseek-r1`, `claude-4.5-sonnet`
    *   **Chat Kecepatan Tinggi**: `gemini-3-flash`, `grok-4.1-fast-non-reasoning`
    *   **Spesialis**: `deepseek-ocr`, `neosantara-gen-2045` (Generasi Gambar)

    <Note>Lihat [Ikhtisar Model](/id/models-overview) untuk harga lengkap dan spesifikasi teknis.</Note>
  </Step>

  <Step title="Pilih SDK kamu">
    Neosantara "triple-compatible." Gunakan SDK yang sesuai dengan alur kerja kamu saat ini.

    | Protokol | Kasus Penggunaan | Base URL |
    | :--- | :--- | :--- |
    | **OpenResponses** | Aplikasi modern, stateful | `https://api.neosantara.xyz/v1` |
    | **OpenAI SDK** | Kompatibilitas legacy | `https://api.neosantara.xyz/v1` |
    | **Anthropic SDK** | Pengguna native Claude | `https://api.neosantara.xyz/anthropic` |
  </Step>

  <Step title="Buat Panggilan Pertama kamu">
    Setelah kamu memiliki `API Key`, kamu dapat mulai membuat permintaan.

    <Tabs>
      <Tab title="OpenResponses (Direkomendasikan)">
        **Direkomendasikan untuk aplikasi berperforma tinggi dan stateful.**

        ```typescript
        const response = await fetch('https://api.neosantara.xyz/v1/responses', {
          method: 'POST',
          headers: {
            'Content-Type': 'application/json',
            'Authorization': 'Bearer KUNCI_NSK_KAMU'
          },
          body: JSON.stringify({
            model: 'nusantara-base',
            input: 'Apa ibu kota Indonesia?',
            store: true
          })
        });
        
        const data = await response.json();
        console.log(data.output_text);
        ```
      </Tab>

      <Tab title="OpenAI SDK">
        **Pengganti langsung untuk integrasi OpenAI.**

        ```python
        from openai import OpenAI

        client = OpenAI(
            api_key="KUNCI_NSK_KAMU",
            base_url="https://api.neosantara.xyz/v1"
        )

        response = client.chat.completions.create(
            model="deepseek-r1",
            messages=[{"role": "user", "content": "1+1=?"}]
        )
        print(response.choices[0].message.content)
        ```
      </Tab>

      <Tab title="Anthropic SDK">
        **Dukungan native untuk format Anthropic Messages.**

        ```python
        import anthropic

        client = anthropic.Anthropic(
            api_key="KUNCI_NSK_KAMU",
            base_url="https://api.neosantara.xyz/anthropic"
        )

        message = client.messages.create(
            model="claude-4.5-sonnet",
            max_tokens=1024,
            messages=[{"role": "user", "content": "Halo!"}]
        )
        print(message.content[0].text)
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>
