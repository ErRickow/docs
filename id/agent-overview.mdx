Anda dapat menggunakan Neosantara AI dengan banyak framework agen AI paling populer. Pilih framework pilihan Anda untuk mempelajari cara meningkatkan agen Anda dengan model sumber terbuka terbaik.

## [CrewAI](/en/crewai)

CrewAI adalah framework sumber terbuka untuk mengorkestrasi sistem agen AI. Ini memungkinkan beberapa agen AI untuk berkolaborasi secara efektif dengan mengambil peran dan bekerja menuju tujuan bersama.

## [PydanticAI](/en/pydantic-ai)

PydanticAI menyediakan ekstraksi data terstruktur dan validasi untuk LLM menggunakan skema Pydantic. Ini memastikan output AI Anda mematuhi format yang ditentukan, membuat integrasi dengan sistem hilir menjadi andal.

## [Agno](/en/agno)

Agno adalah perpustakaan sumber terbuka untuk membuat agen multimodal. Ini mendukung interaksi dengan teks, gambar, audio, dan video sambil tetap agnostik model, memungkinkan Anda menggunakan model apa pun di perpustakaan Neosantara AI dengan integrasi kami.

## [AutoGen(AG2)](/en/autogen)

AutoGen(AG2) adalah framework agen OSS untuk percakapan multi-agen dan otomatisasi alur kerja. Ini memungkinkan pembuatan agen yang dapat disesuaikan yang dapat berinteraksi satu sama lain dan dengan pengguna manusia untuk menyelesaikan tugas yang kompleks.

## [DSPy](/en/dspy)

DSPy adalah framework pemrograman untuk sistem AI algoritmik. Ini menawarkan pendekatan seperti kompiler untuk rekayasa prompt, memungkinkan Anda membuat program model bahasa yang modular, dapat digunakan kembali, dan dapat dioptimalkan.
