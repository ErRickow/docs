## <Icon icon="robot" /> Framework yang Didukung

Pilih framework pilihan Kamu untuk mempelajari cara menyempurnakan agen Kamu dengan Neosantara AI.

<CardGroup cols={3}>
    <Card title="LangChain" icon="link" href="/id/langchain" cta="Pelajari lebih lanjut">
        Framework terpopuler untuk membangun aplikasi LLM. Gunakan integrasi `langchain-openai` kami.
    </Card>
    <Card title="CrewAI" icon="users" href="/id/crewai" cta="Pelajari lebih lanjut">
        Orkestrasi beberapa agen AI untuk berkolaborasi secara efektif dengan asumsi peran dan tujuan bersama.
    </Card>
    <Card title="PydanticAI" icon="shield-check" href="/id/pydantic-ai" cta="Pelajari lebih lanjut">
        Ekstraksi dan validasi data terstruktur menggunakan skema Pydantic untuk output AI yang kamul.
    </Card>
    <Card title="Agno" icon="sparkles" href="/id/agno" cta="Pelajari lebih lanjut">

    </Card>
    <Card title="AutoGen" icon="diagram-project" href="/id/autogen" cta="Pelajari lebih lanjut">

    <Card title="DSPy" icon="microchip" href="/id/dspy" cta="Pelajari lebih lanjut">

</CardGroup>

## <Icon icon="bolt" /> Contoh Cepat

```python Python
from langchain_openai import ChatOpenAI
import os

llm = ChatOpenAI(
    model="nusantara-base",
    openai_api_key=os.environ.get("NAI_API_KEY"),
    openai_api_base="https://api.neosantara.xyz/v1",
)

response = llm.invoke("Buat rencana agen sederhana untuk tugas riset.")
print(response.content)
```
