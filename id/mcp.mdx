Model Context Protocol, atau biasa disebut MCP, adalah standar terbuka yang memungkinkan asisten AI untuk mengakses tool dan sumber data lokal maupun remote secara aman. Neosantara menyediakan server MCP bawaan yang berjalan langsung pada API gateway-nya.

Dengan menghubungkan lingkungan koding lokal kamu ke server MCP Neosantara, kamu dapat mengizinkan tool seperti Cursor atau Claude Desktop untuk memanggil model Neosantara, men-generate gambar, menemukan model yang tersedia, dan mengakses kemampuan agentic secara langsung dari IDE atau chat client kamu.

---

## 1. Buat API Key Khusus MCP

Untuk mengakses server MCP, kamu memerlukan API key khusus MCP. Key Neosantara standar tidak dapat digunakan untuk mengautentikasi endpoint MCP.

<Steps>
  <Step title="Buka Dashboard">
    Buka halaman manajemen API Keys di Dashboard Neosantara di [app.neosantara.xyz/api-keys](https://app.neosantara.xyz/api-keys).
  </Step>
  <Step title="Buat Key">
    Klik tombol **Buat Key** atau **Buat MCP Key**.
  </Step>
  <Step title="Simpan Key">
    Simpan key yang dihasilkan dengan aman. Key kamu akan memiliki awalan `nsk_mcp_`.
  </Step>
</Steps>

<Warning>
  API Key MCP hanya ditampilkan satu kali saat pertama kali dibuat. Simpan dengan aman dan jangan bagikan kepada siapa pun.
</Warning>

---

## 2. Tool yang Didukung

Setelah terhubung, server MCP Neosantara menyediakan tool-tool berikut untuk asisten AI kamu:

| Nama Tool | Deskripsi |
| --- | --- |
| `chat` | Men-generate teks respon menggunakan Responses API Neosantara. Tool ini mendukung payload kaya seperti tools, code interpreter, reasoning, dan instruksi sistem. |
| `list-models` | Menampilkan model bahasa aktif yang tersedia di Neosantara beserta detail kemampuan, context window, dan harganya. |
| `generate-image` | Men-generate gambar berkualitas tinggi dari deskripsi teks. |
| `list-image-models` | Menampilkan model generator gambar yang tersedia, termasuk Google Imagen 4.0 dan Stability AI SDXL. |

## 3. Konfigurasi & Integrasi

Server MCP Neosantara menggunakan protokol transport Server-Sent Events, atau biasa disebut SSE. Ini memungkinkan kamu terhubung langsung melalui URL tanpa perlu mengunduh script executable lokal.

* **URL Endpoint**: `https://api.neosantara.xyz/v1/mcp`
* **Header Otorisasi**: `x-api-key`

### Cursor IDE

Untuk menghubungkan tool Neosantara ke Cursor:

<Steps>
  <Step title="Buka Settings">
    Buka **Cursor Settings** dengan menggunakan pintasan keyboard `Ctrl + ,` di Windows atau `Cmd + ,` di macOS.
  </Step>
  <Step title="Masuk ke Menu MCP">
    Masuk ke menu **Features** lalu pilih **MCP**.
  </Step>
  <Step title="Tambah Server">
    Klik tombol **+ Add New MCP Server**.
  </Step>
  <Step title="Konfigurasi Pengaturan Server">
    Konfigurasikan pengaturan sebagai berikut:
    * **Name**: `neosantara`
    * **Type**: `SSE`
    * **URL**: `https://api.neosantara.xyz/v1/mcp`
  </Step>
  <Step title="Tambah Header Autentikasi">
    Klik **+ Add Header** untuk menambahkan header autentikasi:
    * **Header Name**: `x-api-key`
    * **Header Value**: `API_KEY_NSK_MCP_KAMU`
  </Step>
  <Step title="Simpan dan Verifikasi">
    Klik **Save**. Status server akan berubah menjadi hijau dan menunjukkan 4 tool aktif.
  </Step>
</Steps>

### Claude Desktop

Untuk menghubungkan tool Neosantara ke aplikasi Claude Desktop:

<Steps>
  <Step title="Buka File Konfigurasi">
    Buka file konfigurasi Claude Desktop kamu di salah satu jalur berikut:
    * **macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`
    * **Windows**: `%APPDATA%\Claude\claude_desktop_config.json`
  </Step>
  <Step title="Tambahkan Server MCP Neosantara">
    Tambahkan `neosantara` ke daftar `mcpServers` menggunakan konfigurasi SSE berikut:

    ```json
    {
      "mcpServers": {
        "neosantara": {
          "url": "https://api.neosantara.xyz/v1/mcp",
          "headers": {
            "x-api-key": "API_KEY_NSK_MCP_KAMU"
          }
        }
      }
    }
    ```
  </Step>
  <Step title="Restart Claude Desktop">
    Restart aplikasi Claude Desktop. Kamu akan melihat ikon palu baru di input chat yang menandakan tool aktif.
  </Step>
</Steps>

### Alat CLI dan Agen Terminal

Kamu juga bisa mengintegrasikan server MCP Neosantara dengan asisten koding berbasis CLI dan agen terminal.

#### Gemini CLI

Untuk menambahkan Neosantara sebagai server MCP di Gemini CLI, jalankan perintah berikut:

```bash
gemini mcp add neosantara https://api.neosantara.xyz/v1/mcp -t http -H "x-api-key: KEY_NSK_MCP_KAMU"
```

Untuk memverifikasi koneksi, kamu bisa melihat daftar server MCP yang terkonfigurasi:

```bash
gemini mcp list
```

#### Claude Code

Untuk menambahkan Neosantara sebagai server MCP di Claude Code, jalankan perintah add dengan flag transport HTTP:

```bash
claude mcp add --transport http neosantara https://api.neosantara.xyz/v1/mcp --header "x-api-key: KEY_NSK_MCP_KAMU"
```

Untuk melihat status koneksi, tampilkan daftar server MCP yang aktif:

```bash
claude mcp list
```

#### Codex CLI

Untuk menambahkan Neosantara sebagai server MCP di Codex, gunakan perintah add dan tentukan key kamu:

```bash
codex mcp add neosantara --url https://api.neosantara.xyz/v1/mcp --bearer-token-env-var NAI_MCP_KEY
```

Pastikan kamu sudah meng-export environment variable `NAI_MCP_KEY` yang berisi key MCP Neosantara kamu pada konfigurasi shell.

#### Kiro CLI

Untuk menghubungkan Neosantara ke Kiro CLI, buka file konfigurasi kamu:

* **Jalur**: `~/.kiro/settings/mcp.json`

Tambahkan konfigurasi Neosantara di bawah objek `mcpServers`:

```json
{
  "mcpServers": {
    "neosantara": {
      "url": "https://api.neosantara.xyz/v1/mcp",
      "headers": {
        "x-api-key": "KEY_NSK_MCP_KAMU"
      }
    }
  }
}
```

## 5. Batas Penggunaan

Endpoint MCP memiliki batas throughput dan kuota request harian terpisah yang diberlakukan berdasarkan tier langganan akun kamu. Sebagai contoh, key Free mendapatkan 20 request per menit dan 500 request harian.

Untuk rincian dan detail kuota lengkap per tier, silakan merujuk ke halaman dokumentasi utama [Rate Limit](/id/about/rate-limits#batas-api-mcp).
