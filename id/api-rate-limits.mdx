Halaman ini mendokumentasikan batas tarif untuk endpoint API Neosantara. Batas bervariasi tergantung pada endpoint dan tingkat layanan (tier).

## Batas Global & Per-Tier

- Gratis / Starter: kuota lebih rendah, waktu tunggu lebih lama
- Pro / Enterprise: kuota lebih tinggi dan kapasitas burst

## Batas Per-Endpoint (Contoh)

- Kompatibel OpenAI: kuota per-model, per-kunci
- Kompatibel Anthropic: kuota per-model dengan pertimbangan peralatan (tooling)
- API Respons: kuota yang ditingkatkan dengan konteks stateful

## Panduan Backoff & Percobaan Ulang

- Gunakan exponential backoff dengan jitter.
- Patuhi header `Retry-After` jika disediakan oleh server.

## Pemantauan & Observabilitas

- Log jumlah permintaan, latensi, dan tingkat kesalahan per endpoint.
- Instrumen metrik sisi klien untuk menjaga kuota yang sehat.
