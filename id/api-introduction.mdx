Neosantara menyediakan platform Kecerdasan Buatan berkinerja tinggi dan sadar budaya yang dirancang untuk memberdayakan pengembang dengan kemampuan bahasa, penalaran, dan visi tingkat lanjut. API kami dibangun agar **kompatibel dengan OpenAI**, memastikan pengalaman integrasi yang mulus untuk aplikasi yang sudah ada sambil menawarkan model khusus yang dioptimalkan untuk konteks Indonesia.

## Tujuan

Tujuan utama dari API Neosantara adalah untuk menyediakan tools yang mudah diakses, skalabel, dan cerdas bagi para pengembang untuk membangun aplikasi generasi berikutnya. Baik Kamu sedang membangun agen dukungan pelanggan, asisten penulisan kreatif, atau tools analisis data yang kompleks, API kami memberikan kecerdasan yang Kamu butuhkan dengan nuansa budaya yang penting.

## Langkah Pertama

Ikuti langkah-langkah ini untuk mulai mengintegrasikan Neosantara ke dalam aplikasi Kamu.

<Steps>
  <Step title="Dapatkan Kunci API Kamu">
    Daftar di [Dashboard Neosantara](https://app.neosantara.xyz/dashboard) dan buat kunci API unik Kamu. Kamu akan menerima **kredit gratis di awal** untuk memulai.
  </Step>
  <Step title="Konfigurasi URL Dasar">
    Semua permintaan API harus diarahkan ke URL dasar produksi kami:
    ```bash icon="terminal"
    https://api.neosantara.xyz/v1
    ```
  </Step>
  <Step title="Lakukan permintaan pertama Kamu">
    Gunakan klien HTTP atau SDK favorit Kamu untuk berinteraksi dengan API.
  </Step>
</Steps>

## Otentikasi

Otentikasi ditangani melalui token Bearer. Kamu harus menyertakan kunci API Kamu dalam header `Authorization` pada setiap permintaan.

<CodeGroup>
```bash cURL
curl https://api.neosantara.xyz/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <KUNCI_API_KAMU>" \
  -d '{
    "model": "nusantara-base",
    "messages": [
      {"role": "user", "content": "Halo, apa kabar?"}
    ]
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.neosantara.xyz/v1",
    api_key="<KUNCI_API_KAMU>"
)

response = client.chat.completions.create(
    model="nusantara-base",
    messages=[
        {"role": "user", "content": "Halo, apa kabar?"}
    ]
)

print(response.choices[0].message.content)
```

```javascript Node.js
import OpenAI from "openai";

const openai = new OpenAI({
  baseURL: "https://api.neosantara.xyz/v1",
  apiKey: "<KUNCI_API_KAMU>"
});

const completion = await openai.chat.completions.create({
  messages: [{ role: "user", content: "Halo, apa kabar?" }],
  model: "nusantara-base",
});

console.log(completion.choices[0].message.content);
```

</CodeGroup>
