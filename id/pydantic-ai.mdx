PydanticAI adalah framework agen yang dibuat oleh tim Pydantic untuk menyederhanakan pembangunan aplikasi AI generatif tingkat produksi. Framework ini membawa filosofi desain ergonomis dari FastAPI ke pengembangan agen AI, menawarkan pendekatan yang familiar dan type-safe untuk bekerja dengan model bahasa.

## Instalasi Library

```shell
pip install pydantic-ai -q
```

Atur API Key Neosantara AI Kamu:

```shell
export NAI_API_KEY=***
```

## Contoh Penggunaan

```python icon="python"
from pydantic_ai import Agent
from pydantic_ai.models.openai import OpenAIChatModel
from pydantic_ai.providers.openai import OpenAIProvider
import os

# Hubungkan PydanticAI ke LLM di Neosantara
model = OpenAIChatModel(
    'nusantara-base',
    provider=OpenAIProvider(
        base_url="https://api.neosantara.xyz/v1",
        api_key=os.environ.get("NAI_API_KEY"),
    ),
)

# Setup agen
agent = Agent(
    model,
    system_prompt='Singkat, jawab dalam satu kalimat.',
)

result = agent.run_sync('Dari mana asal kata "hello world"?')  
print(result.output)
```
