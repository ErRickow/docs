Generasi gambar memungkinkan Kamu untuk membuat konten visual unik langsung dari deskripsi tekstual (prompt). Kemampuan ini dapat digunakan untuk aplikasi kreatif, pembuatan konten, desain, dan banyak lagi.

## Penggunaan

Untuk menghasilkan gambar, kirim permintaan POST ke endpoint `/v1/images/generations`. Endpoint ini sepenuhnya kompatibel dengan standar Generasi Gambar OpenAI.

### Endpoint

`POST /v1/images/generations`

<Note>
  Untuk deskripsi parameter terperinci dan untuk menguji API secara langsung, kunjungi [Referensi API](/api-reference/images/create-image-generation).
</Note>

## Kemampuan Tingkat Lanjut

Beberapa model, seperti **Titan Image G1**, mendukung tugas manipulasi gambar tingkat lanjut:

### Image-to-Image (Variasi)

Hasilkan gambar baru berdasarkan gambar yang sudah ada. Berikan gambar asli dalam parameter `image` (sebagai string base64) dan `prompt` yang menjelaskan perubahan atau gaya yang diinginkan.

### Inpainting & Outpainting

*   **Inpainting**: Mengubah area tertentu dari sebuah gambar menggunakan `mask_image`.
*   **Outpainting**: Memperluas gambar melampaui batas-batasnya dengan menyetel `outpainting_mode` ke `"DEFAULT"` atau `"PRECISE"`.

---

## Model Gambar yang Didukung

| ID Model | Terbaik Untuk |
| :--- | :--- |
| `titan-image-g1` | Serbaguna, berkinerja tinggi, mendukung Inpainting/Outpainting |
| `neosantara-gen-2045` | Hasil berkualitas tinggi, artistik, dan sinematik |
| `stable-diffusion-v1-5-img2img` | Generasi gambar-ke-gambar yang efisien dan cepat |

## Contoh Python dan JavaScript (SDK OpenAI)

<CodeGroup>

```python Python
from openai import OpenAI
import os

client = OpenAI(
    api_key=os.environ.get("KUNCI_API_NEOSANTARA_KAMU"),
    base_url="https://api.neosantara.xyz/v1"
)

response = client.images.generate(
    model="neosantara-gen-2045",
    prompt="Sawah di Indonesia yang tenang saat matahari terbit, pencahayaan sinematik.",
    n=1,
    size="1024x1024",
    response_format="url"
)

print(response.data[0].url)
```

```javascript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: "KUNCI_API_NEOSANTARA_KAMU",
  baseURL: "https://api.neosantara.xyz/v1",
});

const response = await client.images.generate({
  model: "titan-image-g1",
  prompt: "Seekor komodo yang agung sedang berjemur di bawah sinar matahari di pantai berbatu, realistis.",
  n: 1,
  size: "1024x1024",
});

console.log(response.data[0].url);
```
