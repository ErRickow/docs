Neosantara API menyediakan endpoint terpadu untuk generasi gambar, yang sepenuhnya kompatibel dengan standar OpenAI.

## Model yang Mendukung

<ModelCards capability="image_generation" />

## Penggunaan

Kirim permintaan `POST` ke endpoint `/v1/images/generations`.

<Note>
  Lihat [Referensi API](/api-reference/images/create-image-generation) untuk detail parameter lengkap.
</Note>

### Manipulasi Gambar Lanjut

Beberapa model mendukung tugas tambahan:
*   **Variasi**: Berikan gambar dalam format base64 pada parameter `image`.
*   **Inpainting**: Ubah area tertentu menggunakan `mask_image`.
*   **Outpainting**: Perluas gambar melampaui batas aslinya menggunakan `outpainting_mode`.

---

## Contoh

<CodeGroup>

```python OpenAI (Python)
from openai import OpenAI
import os

client = OpenAI(
    api_key=os.environ.get("KUNCI_API_NEOSANTARA_KAMU"),
    base_url="https://api.neosantara.xyz/v1"
)

response = client.images.generate(
    model="imagen-4.0-fast",
    prompt="Laboratorium futuristik, berteknologi tinggi, sangat mendetail.",
    n=1,
    size="1024x1024",
    response_format="url"
)

print(response.data[0].url)
```

```javascript OpenAI (Node.js)
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: "KUNCI_API_NEOSANTARA_KAMU",
  baseURL: "https://api.neosantara.xyz/v1",
});

const response = await client.images.generate({
  model: "imagen-4.0-fast",
  prompt: "Laboratorium futuristik, berteknologi tinggi, sangat mendetail.",
  n: 1,
  size: "1024x1024",
});

console.log(response.data[0].url);
```

</CodeGroup>
