---
title: "OCR (Optical Character Recognition)"
description: "Ekstrak teks dari gambar dengan presisi tinggi menggunakan API OCR terpadu Neosantara."
---

Neosantara menyediakan endpoint OCR khusus yang memanfaatkan model-model tercanggih (seperti DeepSeek OCR) untuk mengekstrak teks dari berbagai format gambar, termasuk dokumen, tanda terima, dan catatan tulisan tangan.

## Endpoint

| Metode | Endpoint |
| :--- | :--- |
| `POST` | `/v1/ocr` |

## Contoh Penggunaan

### Ekstrak Teks dari URL Gambar

Kamu dapat memberikan URL langsung ke gambar untuk diproses.

```bash icon="terminal"
curl https://api.neosantara.xyz/v1/ocr \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer KUNCI_API_NEOSANTARA_KAMU" \
  -d '{"image": "https://example.com/receipt.jpg", "image_type": "url"}'
```

```python icon="python"
import requests

url = "https://api.neosantara.xyz/v1/ocr"
headers = {
    "Authorization": "Bearer KUNCI_API_NEOSANTARA_KAMU",
    "Content-Type": "application/json"
}
payload = {
    "image": "https://example.com/receipt.jpg",
    "image_type": "url"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json()["choices"][0]["text"])
```

### Ekstrak Teks dari Base64

Untuk gambar lokal, Kamu dapat menyandikannya sebagai string Base64.

```bash icon="terminal"
curl https://api.neosantara.xyz/v1/ocr \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer KUNCI_API_NEOSANTARA_KAMU" \
  -d '{"image": "data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAAAQ...", "image_type": "base64"}'
```

## Parameter

| Parameter | Tipe | Wajib | Default | Deskripsi |
| :--- | :--- | :--- | :--- | :--- |
| `image` | `string` | Ya | - | URL gambar atau string data Base64. |
| `image_type` | `string` | Tidak | `url` | Jenis input gambar: `url` atau `base64`. |
| `return_image` | `boolean` | Tidak | `false` | Jika `true`, mengembalikan gambar yang diproses dengan kotak pembatas OCR (jika didukung oleh model). |

## Format Respons

API mengembalikan objek JSON yang berisi teks yang diekstrak dan metadata.

```json icon="code"
{
  "id": "ocr-abc123def456",
  "object": "ocr.completion",
  "choices": [
    {
      "index": 0,
      "text": "KONTEN TEKS DI SINI",
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "total_tokens": 50
  }
}
```
