---
title: "OCR (Optical Character Recognition)"
description: "Ekstrak teks dari gambar dengan presisi tinggi menggunakan API OCR terpadu dari Neosantara."
---

Neosantara AI menyediakan endpoint OCR khusus yang memanfaatkan model tercanggih (seperti DeepSeek OCR) untuk mengekstrak teks dari berbagai format gambar, termasuk dokumen, struk belanja, dan catatan tulisan tangan.

## Endpoint

| Method | Endpoint | Deskripsi |
| :--- | :--- | :--- |
| `POST` | `/v1/ocr` | Endpoint ekstraksi OCR terpadu |

## Contoh Penggunaan

### Ekstrak Teks dari URL Gambar

Kamu dapat memberikan URL langsung ke gambar untuk diproses.

<CodeGroup>
```bash cURL
curl https://api.neosantara.xyz/v1/ocr \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <KUNCI_API_KAMU>" \
  -d '{
    "image": "https://example.com/receipt.jpg",
    "image_type": "url"
  }'
```

```python Python
import requests

url = "https://api.neosantara.xyz/v1/ocr"
headers = {
    "Authorization": "Bearer <KUNCI_API_KAMU>",
    "Content-Type": "application/json"
}
payload = {
    "image": "https://example.com/receipt.jpg",
    "image_type": "url"
}

response = requests.post(url, json=payload, headers=headers)
print(response.json()["extracted_text"])
```

</CodeGroup>

### Ekstrak Teks dari Base64

Untuk gambar lokal, Kamu dapat menyandikannya sebagai string Base64.

<CodeGroup>
```bash cURL
curl https://api.neosantara.xyz/v1/ocr \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <KUNCI_API_KAMU>" \
  -d '{
    "image": "data:image/png;base64,iVBORw0KGgoAAAANSUhEUg...",
    "image_type": "base64"
  }'
```
</CodeGroup>

## Parameter

| Parameter | Tipe | Wajib | Default | Deskripsi |
| :--- | :--- | :--- | :--- | :--- |
| `image` | `string` | Ya | - | URL gambar atau string data Base64. |
| `image_type` | `string` | Tidak | `url` | Tipe input gambar: `url` atau `base64`. |
| `return_image` | `boolean` | Tidak | `false` | Jika `true`, mengembalikan gambar yang telah diproses dengan kotak pembatas OCR (jika didukung oleh model). |

## Format Respons

API mengembalikan objek JSON yang berisi teks hasil ekstraksi dan metadata.

```json icon="code"
{
  "extracted_text": "ISI TEKS DI SINI",
  "status": "success",
  "model": "deepseek-ocr",
  "usage": {
    "tokens": 50
  }
}
```
