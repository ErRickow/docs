Neosantara AI menyediakan antarmuka terpadu untuk pembuatan video, yang ditenagai oleh model **GLM-4 Video** (CogVideoX). Kemampuan ini memungkinkan Anda membuat video sinematik beresolusi tinggi langsung dari deskripsi teks atau dengan menyediakan gambar sebagai referensi.

-----

### Cara Menggunakan API

Pembuatan video adalah proses asinkron. Anda pertama-tama mengirimkan pekerjaan (*job*), lalu melakukan pengecekan statusnya secara berkala.

#### **1. Kirim Pekerjaan Video**

Kirim permintaan `POST` ke `/v1/videos`.

<CodeGroup>
```json cURL icon="terminal"
curl -X POST "https://api.neosantara.xyz/v1/videos" \
-H "Authorization: Bearer <NAI_API_KEY>" \
-H "Content-Type: application/json" \
-d 
{
  "model": "cogvideox",
  "prompt": "Matahari terbenam yang indah di atas kepulauan Indonesia, pencahayaan sinematik, 4k",
  "video_style": "cinematic"
}
```
</CodeGroup>

**Parameter:**
- `model`: Wajib. Gunakan `cogvideox` atau `cogvideox-pro`.
- `prompt`: Wajib. Deskripsi rinci tentang video yang ingin Anda buat (maks 2000 karakter).
- `input_reference`: Opsional. URL gambar untuk digunakan sebagai titik awal (Image-to-Video).
- `video_style`: Opsional. misal, `cinematic`, `anime`, `realistic`.

#### **2. Cek Status Pekerjaan**

Gunakan `id` yang dikembalikan dari pengiriman untuk mengecek kemajuan.

<CodeGroup>
```json cURL icon="terminal"
curl "https://api.neosantara.xyz/v1/videos/<JOB_ID>" \
-H "Authorization: Bearer <NAI_API_KEY>"
```
</CodeGroup>

#### **3. Unduh Konten**

Setelah status menjadi `completed`, ambil file videonya.

<CodeGroup>
```json cURL icon="terminal"
curl "https://api.neosantara.xyz/v1/videos/<JOB_ID>/content" \
-H "Authorization: Bearer <NAI_API_KEY>"
```
</CodeGroup>

-----

### Harga & Kuota

Pembuatan video adalah tugas komputasi tinggi dan ditagih per video yang dihasilkan.

| Model | Harga (IDR) | Ekuivalen Token |
| :--- | :--- | :--- |
| **cogvideox** | Rp 15,000 | 7,500 Token |
| **cogvideox-pro** | Rp 25,000 | 12,500 Token |

<Note>
  Jika Anda dalam mode **Pay-As-You-Go**, biaya akan dipotong dari saldo Anda. Untuk pengguna langganan, ini akan dihitung terhadap kuota token harian/bulanan Anda menggunakan nilai ekuivalen token.
</Note>

-----

### Praktik Terbaik

*   **Jadilah Deskriptif**: Deskripsikan subjek, aksi, pencahayaan, dan pergerakan kamera.
*   **Resolusi**: Model saat ini mendukung output hingga 1024x1024 atau 720p tergantung pada variannya.
*   **Panjang Prompt**: Jaga prompt di bawah 2000 karakter untuk hasil yang optimal.
