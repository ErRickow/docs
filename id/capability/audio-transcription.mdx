Neosantara AI menyediakan endpoint berperforma tinggi untuk mentranskripsi file audio. Ditenagai oleh model seperti **Whisper Large v3 Turbo**, gateway kami menawarkan transkripsi instan dengan presisi mendalam, mendukung berbagai bahasa dan format.

## Cara Kerja

Kamu mengunggah file audio (MP3, WAV, M4A, dll.) ke endpoint terpadu kami, dan model akan mengubah kata-kata lisan menjadi teks tertulis. Kamu dapat memilih antara respon teks sederhana atau struktur JSON detail dengan timestamp.

API Neosantara AI menyediakan endpoint [`/v1/audio/transcriptions`](/api-reference/audio/transcription) untuk tujuan ini.

## Penggunaan

Untuk mentranskripsi audio, kirim permintaan `multipart/form-data` ke endpoint `/v1/audio/transcriptions`.

### Endpoint

`POST /v1/audio/transcriptions`

### Parameter yang Didukung

| Parameter | Tipe | Wajib | Default | Deskripsi |
| :--- | :--- | :--- | :--- | :--- |
| **`file`** | File | Ya | - | Objek file audio yang akan ditranskripsi. Didukung: `mp3`, `mp4`, `mpeg`, `mpga`, `m4a`, `wav`, `webm`. |
| **`model`** | String | Ya | - | ID model yang digunakan. Saat ini didukung: `whisper-large-v3-turbo`. |
| **`language`** | String | Tidak | - | Bahasa input audio (format ISO-639-1). |
| **`prompt`** | String | Tidak | - | Teks opsional untuk memandu gaya model atau melanjutkan segmen sebelumnya. |
| **`response_format`** | String | Tidak | `json` | Format hasil transkripsi. Opsi: `json`, `text`, `verbose_json`. |
| **`temperature`** | Number | Tidak | `0` | Suhu sampling (0 hingga 1). |

## Contoh Python dan JavaScript (OpenAI SDK)

Karena Neosantara kompatibel dengan OpenAI, Kamu dapat menggunakan SDK resmi untuk mentranskripsi audio dengan mulus.

<CodeGroup>

```python Python
import os
from openai import OpenAI

# Inisialisasi klien OpenAI
client = OpenAI(
    api_key=os.environ.get("NAI_API_KEY"),
    base_url="https://api.neosantara.xyz/v1",
)

def transkripsi_audio(file_path: str):
    """
    Mentranskripsi file audio menggunakan Neosantara AI.
    """
    print(f"Memproses file: {file_path}...")

    try:
        with open(file_path, "rb") as audio_file:
            transcription = client.audio.transcriptions.create(
                model="whisper-large-v3-turbo",
                file=audio_file,
                response_format="text"
            )
        
        print("\nHasil Transkripsi:")
        print(transcription)
        return transcription

    except Exception as e:
        print(f"\nTerjadi kesalahan saat transkripsi: {e}")
        return None

if __name__ == "__main__":
    transkripsi_audio("rekaman_rapat.mp3")
```

```javascript Node.js
import fs from "fs";
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: "https://api.neosantara.xyz/v1",
});

async function main() {
  const transcription = await client.audio.transcriptions.create({
    file: fs.createReadStream("speech.mp3"),
    model: "whisper-large-v3-turbo",
  });

  console.log(transcription.text);
}

main();
```
