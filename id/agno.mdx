---
title: "Agno"
description: "> Menggunakan Agno dengan Neosantara."
---

Agno adalah library sumber terbuka untuk membuat agen multimodal. Agno mendukung interaksi dengan teks, gambar, audio, dan video sambil tetap bersifat agnostik model, memungkinkan kamu menggunakan model apa pun di perpustakaan Neosantara dengan integrasi asli kami.

## Instal Perpustakaan

```bash icon="terminal"
pip install -U agno ddgs -q
```

## Autentikasi

Setel variabel lingkungan `NEOSANTARA_API_KEY` kamu.

```shell
export NEOSANTARA_API_KEY=***
```

## Contoh

Di bawah ini adalah agen sederhana menggunakan integrasi asli Neosantara dengan akses ke pencarian web.

```python icon="python"
from agno.agent import Agent
from agno.models.neosantara import Neosantara
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(
    model=Neosantara(id="nusantara-base"),
    tools=[DuckDuckGoTools()],
    markdown=True
)
agent.print_response("Apa yang sedang terjadi di Indonesia?", stream=True)
```
