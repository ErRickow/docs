---

Agno adalah library open-source untuk membuat agen multimodal. Agno mendukung interaksi dengan teks, gambar, audio, dan video sambil tetap bersifat model-agnostic, memungkinkan Kamu menggunakan model apa pun dalam library Neosantara AI dengan integrasi kami.

## Instalasi Library

```bash
pip install -U agno ddgs -q
```

## Autentikasi

Atur variabel lingkungan `NAI_API_KEY` Kamu.

```shell
export NAI_API_KEY=***
```

## Contoh Penggunaan

Di bawah ini adalah contoh agen sederhana dengan akses ke pencarian web.

```python
from agno.agent import Agent
from agno.models.openai.like import OpenAILike
from agno.tools.duckduckgo import DuckDuckGoTools
import os

agent = Agent(
    model=OpenAILike(
        id="nusantara-base",
        api_key=os.environ.get("NAI_API_KEY"),
        base_url="https://api.neosantara.xyz/v1"
    ),
    tools=[DuckDuckGoTools()],
    markdown=True
)
agent.print_response("Apa yang sedang terjadi di Indonesia?", stream=True)
```
