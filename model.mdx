## Model Selection Guide

Gunakan panduan ini untuk memilih model yang tepat:

1. **Konten Bahasa Indonesia**:
   - Utamakan: nusantara-base, archipelago-7b, bahasa-llm

2. **Pemrosesan Cepat**:
   - Utamakan: bahasa-llm, gemini-1.5-flash

3. **Pencarian & Analisis**:
   - Utamakan: command-r-plus (web search)
   - Embedding: nusa-embedding-0001

4. **Function Calling**:
   - Utamakan: nusantara-base, bahasa-llm

## Model Capabilities

| Capability | Description |
|------------|-------------|
| function_calling | Kemampuan memanggil fungsi eksternal |
| json_mode | Output dalam format JSON terstruktur |
| web_search | Kemampuan mencari informasi dari web |
| streaming | Mendukung response streaming |
