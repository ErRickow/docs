Neosantara API provides a unified endpoint for image generation, fully compatible with the OpenAI standard.

## Supported Models

<ModelCards capability="image_generation" />

## Usage

Send a `POST` request to `/v1/images/generations`.

<Note>
  See the [API Reference](/api-reference/images/create-image-generation) for full parameter details.
</Note>

### Image-to-Image & Inpainting

Some models support advanced tasks:
*   **Variation**: Provide a base64 image in the `image` parameter.
*   **Inpainting**: Modify specific areas using a `mask_image`.
*   **Outpainting**: Extend images using `outpainting_mode`.

## Examples

<CodeGroup>

```python Python
from openai import OpenAI
import os

client = OpenAI(
    api_key=os.environ.get("YOUR_NEOSANTARA_API_KEY"),
    base_url="https://api.neosantara.xyz/v1"
)

response = client.images.generate(
    model="neosantara-gen-2045",
    prompt="A serene Indonesian rice paddy field at sunrise, cinematic lighting.",
    n=1,
    size="1024x1024",
    response_format="url"
)

print(response.data[0].url)
```

```javascript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: "YOUR_NEOSANTARA_API_KEY",
  baseURL: "https://api.neosantara.xyz/v1",
});

const response = await client.images.generate({
  model: "titan-image-g1",
  prompt: "A majestic Komodo dragon basking in the sun on a rocky beach, realistic.",
  n: 1,
  size: "1024x1024",
});

console.log(response.data[0].url);
```
