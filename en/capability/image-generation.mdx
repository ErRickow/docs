Image generation allows you to create unique visual content directly from textual descriptions (prompts). This capability can be used for creative applications, content creation, design, and more.

## Usage

To generate images, send a POST request to the `/v1/images/generations` endpoint. This endpoint is fully compatible with the OpenAI Image Generation standard.

### Endpoint

`POST /v1/images/generations`

<Note>
  For detailed parameter descriptions and to test the API directly, visit the [API Reference](/api-reference/images/create-image-generation).
</Note>

## Advanced Capabilities

Some models, like **Titan Image G1**, support advanced image manipulation tasks:

### Image-to-Image (Variation)

Generate a new image based on an existing one. Provide the original image in the `image` parameter (as a base64 string) and a `prompt` describing the desired changes or style.

### Inpainting & Outpainting

*   **Inpainting**: Modify specific areas of an image using a `mask_image`.
*   **Outpainting**: Extend an image beyond its borders by setting `outpainting_mode` to `"DEFAULT"` or `"PRECISE"`.

## Supported Image Models

| Model ID | Best For |
| :--- | :--- |
| `titan-image-g1` | Versatile, high-performance, supports Inpainting/Outpainting |
| `neosantara-gen-2045` | High-quality, artistic, and cinematic outputs |
| `stable-diffusion-v1-5-img2img` | Efficient and fast image-to-image generation |

## Python and JavaScript Examples (OpenAI SDK)

<CodeGroup>

```python Python
from openai import OpenAI
import os

client = OpenAI(
    api_key=os.environ.get("YOUR_NEOSANTARA_API_KEY"),
    base_url="https://api.neosantara.xyz/v1"
)

response = client.images.generate(
    model="neosantara-gen-2045",
    prompt="A serene Indonesian rice paddy field at sunrise, cinematic lighting.",
    n=1,
    size="1024x1024",
    response_format="url"
)

print(response.data[0].url)
```

```javascript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: "YOUR_NEOSANTARA_API_KEY",
  baseURL: "https://api.neosantara.xyz/v1",
});

const response = await client.images.generate({
  model: "titan-image-g1",
  prompt: "A majestic Komodo dragon basking in the sun on a rocky beach, realistic.",
  n: 1,
  size: "1024x1024",
});

console.log(response.data[0].url);
```
