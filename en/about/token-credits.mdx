## Hybrid Billing System

Neosantara AI uses a flexible **Hybrid Billing System** that combines subscription quotas with a Pay-As-You-Go model.

1.  **Subscription Quota:** Each plan comes with a set amount of "Free Tokens" (Quota) that resets monthly.
2.  **Pay-As-You-Go:** Once your quota is exhausted, you can continue using the API by topping up your **Credit Balance**.
3.  **Automatic Switching:** If you exceed your Monthly Token Limit, your account automatically switches to **Pay-As-You-Go Mode** permanently to ensure uninterrupted service. **We grant you a Rp 10,000 bonus balance upon this switch!**

### Base Pricing

For Pay-As-You-Go usage, the base rate for standard models is approximately:

-   **Rp 149,999 per 1 million tokens**
-   **~Rp 0.15 per token**

### How Costs Are Calculated

$$Price = (\text{Input Tokens} \times \text{Input Price}) + (\text{Output Tokens} \times \text{Output Price})$$

**Example:**
Using `nusantara-base` (Input: Rp 300/1M, Output: Rp 1,500/1M) for a request with 500 input tokens and 200 output tokens:

1.  Input Cost: $500 / 1,000,000 \times 300 = \text{Rp } 0.15$
2.  Output Cost: $200 / 1,000,000 \times 1,500 = \text{Rp } 0.30$
3.  **Total:** Rp 0.45
