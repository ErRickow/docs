Neosantara uses pay-as-you-go billing. Requests are charged from your credit balance based on the selected model, request type, and usage units such as tokens, generated images, audio duration, or video duration.

<Note>
  Per-model prices are sourced from the live model catalog. Browse [Models Overview](/en/models-overview) or open a generated model page for current pricing units.
</Note>

## Get started

1. Create an account in the [Neosantara app](https://app.neosantara.xyz).
2. Verify your email to receive the current new-account credit benefit when available.
3. Create an API key from [API Keys](https://app.neosantara.xyz/api-keys).
4. Monitor balance and usage from the [Usage page](https://app.neosantara.xyz/usage).

## How billing works

| Usage type | Billing unit | Where to check |
| --- | --- | --- |
| Text, chat, reasoning, tools | Input and output tokens | Generated model pages and response usage metadata |
| Embeddings | Input tokens | [Embeddings models](/models/nusa-embedding-0001) |
| Image generation | Image size or generation unit | Generated image model pages |
| Audio transcription | Audio duration | Generated audio model pages |
| Video generation | Duration or provider-specific unit | Generated video model pages |

Some models are priced in IDR, while provider-priced models may be shown in USD and converted at billing time.

## Batch discount

The Batch API is designed for asynchronous high-volume work that does not need an immediate response.

| Usage type | Rate adjustment |
| --- | --- |
| Batch input | 50% of standard input price |
| Batch output | 50% of standard output price |

<Note>
  The discount is applied automatically for requests sent through `/v1/batches`.
</Note>

## Cache read

For models that support prompt caching, cached input may be billed at a separate cache-read rate. If a model page does not list a cache-read price, standard input pricing applies.

## Tool usage

Tool definitions, tool calls, and tool results are billed as normal input and output tokens for the selected model. Neosantara does not add a separate orchestration fee for tool calling.

## Throughput tiers

Throughput limits are tied to your account tier. Tier upgrades are applied automatically based on lifetime deposit history.

| Tier | Min. cumulative deposit | RPM | ITPM | OTPM |
| --- | --- | --- | --- | --- |
| Free | Rp 0 | 3 | 15,000 | 2,000 |
| Basic | Rp 85,000 | 50 | 50,000 | 10,000 |
| Standard | Rp 670,000 | 1,000 | 450,000 | 90,000 |
| Pro | Rp 3,350,000 | 2,000 | 1,000,000 | 200,000 |
| Enterprise | Rp 6,700,000 | 4,000 | 4,000,000 | 800,000 |

See [Rate Limits](/en/about/rate-limits) for response headers, retry behavior, and limit handling.

## Balance management

Current balance and usage are available in the dashboard. API responses also include usage and billing metadata where supported.

To add balance, open the [Billing page](https://app.neosantara.xyz/billing), choose **Top Up Balance**, and complete payment through the available payment methods.
