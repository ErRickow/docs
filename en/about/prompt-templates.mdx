---
title: "Prompt Templates"
description: "Create and reuse dynamic prompts with variables, dedicated exclusively to the Responses API."
---

Prompt templates allow you to save and manage complex instructions on the Neosantara AI platform. Instead of sending long prompts with every API call, you can reference a template ID and inject variables dynamically.

<Warning>
  **Exclusive to Responses API**: Prompt templates are a dedicated feature of the `/v1/responses` endpoint. They are **not** supported by the legacy `/v1/chat/completions` endpoint.
</Warning>

## Creating a Template

You can manage your templates directly from the dashboard:

1.  Navigate to your **[Dashboard](https://app.neosantara.xyz/dashboard)**.
2.  Select the **"Prompt Templates"** section.
3.  Define your **ID** (e.g., `legal-analyzer`) and **Content**.
4.  Use `{{variable_name}}` syntax for dynamic placeholders.

**Example Content:**

```text
You are a legal expert. Analyze the following document: {{document_text}}. 
Focus specifically on {{focus_area}}.
```

## Using Templates with OpenAI SDK

Neosantara AI's Responses API is compatible with the latest OpenAI SDK structure. You can use the dedicated `responses` namespace to call your templates.

<CodeGroup>

```python Python
from openai import OpenAI
import os

client = OpenAI(
    api_key="YOUR_NEOSANTARA_API_KEY",
    base_url="https://api.neosantara.xyz/v1"
)

# Use the dedicated responses namespace
response = client.responses.create(
    model="claude-4.5-sonnet",
    prompt={
        "id": "legal-analyzer",
        "variables": {
            "document_text": "This is a sample contract...",
            "focus_area": "liability clauses"
        }
    }
)

print(response.output_text)
```

```javascript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'YOUR_NEOSANTARA_API_KEY',
  baseURL: 'https://api.neosantara.xyz/v1'
});

// Access the responses API directly
const response = await client.responses.create({
  model: 'claude-4.5-sonnet',
  prompt: {
    id: 'legal-analyzer',
    variables: {
      document_text: 'This is a sample contract...', 
      focus_area: 'liability clauses'
    }
  }
});

console.log(response.output_text);
```

```bash cURL
curl https://api.neosantara.xyz/v1/responses \
  -H "Authorization: Bearer YOUR_NEOSANTARA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-4.5-sonnet",
    "prompt": {
      "id": "legal-analyzer",
      "variables": {
        "document_text": "...",
        "focus_area": "..."
      }
    }
  }'
```

## Why use Prompt Templates?

| Feature | Standard Chat | Prompt Templates |
| :--- | :--- | :--- |
| **Reusability** | Must resend full prompt | Reference by ID |
| **Security** | Prompts visible in code | Logic hidden in Dashboard |
| **Payload Size** | Larger (full text) | Smaller (ID + variables) |
| **Endpoint** | `/v1/chat/completions` | `/v1/responses` |
