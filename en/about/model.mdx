## Model Selection Guide

Use this guide to choose the right model for your needs:

1. **Indonesian Content & Local Context**:
    - Prioritize: nusantara-base, archipelago-7b, bahasa-llm

2. **Fast & Efficient Processing**:
    - Prioritize: bahasa-llm, gemini-1.5-flash, gemini-2.0-flash, gemma2-9b-it

3. **Information Retrieval & In-depth Analysis**:
    - Prioritize: command-r-plus (for web search), gemini-1.5-pro (for analysis)

4. **Function Calling**:
    - Prioritize: nusantara-base, bahasa-llm, gemini-1.5-flash, gemini-1.5-pro, gpt-3.5-turbo, gpt-4o-mini, firefunction-v1, llama-3.3-70b-instruct

5. **Image Generation**:
    - Use: neosantara-gen-2045, replicate-stable-diffusion-xl

6. **Image Analysis (Vision)**:
    - Use: vision-emas-2045

7. **Embedding (Text Mapping)**:
    - Use: nusa-embedding-0001, together-embed-v2-L

## Model Capabilities

Here is a list of the main capabilities supported by Neosantara AI models:

| Capability | Description |
|------------|-------------|
| function_calling | The ability of the model to call external functions |
| json_mode | The ability of the model to generate output in structured JSON format |
| web_search | The ability of the model to search for information from the web |
| streaming | Supports streaming responses (gradual output) |
| image_generation | The ability of the model to create images from text prompts |
| vision | The ability of the model to analyze and understand content from images |
| embedding | The ability of the model to convert text into numerical vector representations |
