---
title: 'Authentication'
description: 'How to authenticate against Neosantara API endpoints.'
---

Neosantara API supports standard authentication methods across endpoints.

## Methods

- Bearer Token (API Key in Authorization header)
- API Key header (e.g., `x-api-key`)
- Environment variables (recommended for local development)

## How to Use

- Set Authorization header: `Authorization: Bearer <YOUR_API_KEY>`
- Or provide API key via header like `x-api-key: <KEY>` depending on client library.
- For browser-based apps, use a secure backend to proxy requests and keep keys secret.

## Examples

Python

```python
import requests
headers = {"Authorization": "Bearer <YOUR_API_KEY>"}
payload = {"model": "nusantara-base", "messages": [{"role": "user", "content": "Hello"}]}
resp = requests.post("https://api.neosantara.xyz/v1/chat/completions", json=payload, headers=headers)
```

JavaScript

```js
const payload = {
  model: "nusantara-base",
  messages: [{ role: "user", content: "Hello" }]
};

const res = await fetch("https://api.neosantara.xyz/v1/chat/completions", {
  method: 'POST',
  headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer <YOUR_API_KEY>' },
  body: JSON.stringify(payload)
})
```
