## Getting Started

Follow these steps to make your first API call to Neosantara AI. Our platform acts as a **unified gateway**, allowing you to access various SOTA models from different providers using a single API key and SDK.

<Steps>
  <Step title="Get API Key">
    * Access [Neosantara App](https://app.neosantara.xyz/signup) to Register or Login.
    * Access the [Billing Page](https://app.neosantara.xyz/billing) to top up your balance if needed.
    * Create an API Key in the [API Keys](https://app.neosantara.xyz/api-keys) management page.
    * Copy your API Key securely.

    <CardGroup cols={2}>
      <Card title="Neosantara Dashboard" icon="link" href="https://app.neosantara.xyz/dashboard">
        Manage your account, billing, and usage statistics.
      </Card>

      <Card title="API Keys Management" icon="key" href="https://app.neosantara.xyz/api-keys">
        Create and manage your unique API access keys.
      </Card>
    </CardGroup>
  </Step>

  <Step title="Choose Your Model">
    Access the world's best models through a single interface. Switch between providers by simply changing the `model` parameter. See [Models & Pricing](/en/about/token-credits) for the full list.

    <CardGroup cols={2}>
      <Card title="Global Flagships" icon="sparkles" href="/en/models-overview">
        Access **Claude 4.5 Sonnet**, **Gemini 3 Flash**, and **Grok 4.1** via a single integration.
      </Card>

      <Card title="Coding & Logic" icon="code" href="/models/qwen3-32b">
        High-performance reasoning with **Qwen 3 32B** and **Llama 3.3 Nemotron**.
      </Card>

      <Card title="Multimodal & Visi" icon="eye" href="/en/capability/ocr">
        Generate images with **Gen-2045** or extract text with **DeepSeek OCR**.
      </Card>

      <Card title="Regional Experts" icon="earth-asia" href="/models/archipelago-70b">
        Optimized for Indonesian context with **Archipelago-70b** and **Sea-Lion**.
      </Card>
    </CardGroup>
  </Step>

  <Step title="Choose Calling Method">
    Neosantara is designed to be a multi-compatible gateway. Use the official SDK that fits your existing ecosystem.

    <CardGroup cols={3}>
      <Card title="OpenAI SDK" icon="openai" href="/en/sdk/openai-compatible">
        Drop-in replacement for any OpenAI-based application.
      </Card>

      <Card title="Anthropic SDK" icon="bolt" href="/en/sdk/anthropic-compatible">
        Native support for the Anthropic Messages API format.
      </Card>

      <Card title="API Reference" icon="book" href="/api-reference">
        Complete documentation for direct REST API calls.
      </Card>
    </CardGroup>
  </Step>

  <Step title="Make Your First Call">
    Once you have your `API Key` and selected a model, you can start making requests. Here are examples using popular methods:

    <Tabs>
      <Tab title="Python SDK">
        **Install SDK**

        <CodeGroup>
        ```bash pip
        pip install --upgrade openai
        ```
        </CodeGroup>

        **Usage Example**

        ```python
        from openai import OpenAI
        import os

        client = OpenAI(
            api_key="YOUR_NEOSANTARA_API_KEY",
            base_url="https://api.neosantara.xyz/v1"
        )

        # You can use ANY model available on the platform here
        completion = client.chat.completions.create(
            model="gemini-3-flash",
            messages=[
                {"role": "system", "content": "You are a helpful assistant."},
                {"role": "user", "content": "What are the benefits of an LLM Gateway?"}
            ]
        )

        print(completion.choices[0].message.content)
        ```
      </Tab>

      <Tab title="Node.js SDK">
        **Install SDK**

        <CodeGroup>
        ```bash npm
        npm install openai
        ```
        </CodeGroup>

        **Usage Example**

        ```javascript
        import OpenAI from "openai";

        const client = new OpenAI({
            apiKey: "YOUR_NEOSANTARA_API_KEY",
            baseURL: "https://api.neosantara.xyz/v1"
        });

        async function main() {
            const response = await client.responses.create({
                model: "claude-4.5-sonnet",
                input: "Explain quantum computing in one sentence.",
                store: true
            });

            console.log(response.output_text);
        }

        main();
        ```
      </Tab>

      <Tab title="cURL">
        **Usage Example**

        ```bash
        curl https://api.neosantara.xyz/v1/chat/completions \
          -H "Content-Type: application/json" \
          -H "Authorization: Bearer YOUR_NEOSANTARA_API_KEY" \
          -d '{
            "model": "grok-4.1-fast-non-reasoning",
            "messages": [
              {"role": "user", "content": "Tell me a joke about developers."}
            ]
          }'
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>
