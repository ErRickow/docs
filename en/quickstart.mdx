## Getting Started

Neosantara acts as a **unified gateway**, allowing you to access various state-of-the-art models from different providers using a single API key and your choice of SDK.

<Steps>
  <Step title="Get your API Key">
    1.  Sign up or Login at the [Neosantara App](https://app.neosantara.xyz).
    2.  Go to the [Dashboard](https://app.neosantara.xyz/dashboard) to view your initial free credits.
    3.  Create a new key in the [API Keys](https://app.neosantara.xyz/api-keys) section.
    4.  Copy your key (it starts with `nsk_`).

    <CardGroup cols={2}>
      <Card title="Dashboard" icon="link" href="https://app.neosantara.xyz/dashboard">
        Manage account, billing, and stats.
      </Card>
      <Card title="API Keys" icon="key" href="https://app.neosantara.xyz/api-keys">
        Manage your access credentials.
      </Card>
    </CardGroup>
  </Step>

  <Step title="Choose Your Model">
    Access the world's best models through a single interface. Switch models by simply changing the `model` string. 

    *   **Flagship Reasoning**: `deepseek-r1`, `claude-4.5-sonnet`
    *   **High-Speed Chat**: `gemini-3-flash`, `grok-4.1-fast-non-reasoning`
    *   **Specialized**: `deepseek-ocr`, `neosantara-gen-2045` (Image Gen)

    <Note>See the [Models Overview](/en/models-overview) for full pricing and technical specs.</Note>
  </Step>

  <Step title="Pick Your SDK">
    Neosantara is "triple-compatible." Use the SDK that fits your existing workflow.

    | Protocol | Use Case | Base URL |
    | :--- | :--- | :--- |
    | **OpenResponses** | Stateful, modern apps | `https://api.neosantara.xyz/v1` |
    | **OpenAI SDK** | Legacy compatibility | `https://api.neosantara.xyz/v1` |
    | **Anthropic SDK** | native Claude users | `https://api.neosantara.xyz/anthropic` |
  </Step>

  <Step title="Make Your First Call">
    Once you have your `API Key`, you can start making requests.

    <Tabs>
      <Tab title="OpenResponses (Recommended)">
        **Recommended for stateful, high-performance apps.**

        ```typescript
        const response = await fetch('https://api.neosantara.xyz/v1/responses', {
          method: 'POST',
          headers: {
            'Content-Type': 'application/json',
            'Authorization': 'Bearer YOUR_NSK_KEY'
          },
          body: JSON.stringify({
            model: 'nusantara-base',
            input: 'What is the capital of Indonesia?',
            store: true
          })
        });
        
        const data = await response.json();
        console.log(data.output_text);
        ```
      </Tab>

      <Tab title="OpenAI SDK">
        **Drop-in replacement for OpenAI integrations.**

        ```python
        from openai import OpenAI

        client = OpenAI(
            api_key="YOUR_NSK_KEY",
            base_url="https://api.neosantara.xyz/v1"
        )

        response = client.chat.completions.create(
            model="deepseek-r1",
            messages=[{"role": "user", "content": "1+1=?"}]
        )
        print(response.choices[0].message.content)
        ```
      </Tab>

      <Tab title="Anthropic SDK">
        **Native support for Anthropic Messages format.**

        ```python
        import anthropic

        client = anthropic.Anthropic(
            api_key="YOUR_NSK_KEY",
            base_url="https://api.neosantara.xyz/anthropic"
        )

        message = client.messages.create(
            model="claude-4.5-sonnet",
            max_tokens=1024,
            messages=[{"role": "user", "content": "Hello!"}]
        )
        print(message.content[0].text)
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>
