## Which one should I choose?

- **OpenAI Compatible**: Best if you already have an OpenAI-based application or are using universal libraries like LangChain or LlamaIndex.
- **Anthropic Compatible**: Ideal if you are building with Claude-specific tools like Claude Code or prefer the Anthropic Messages structure.
