---
title: "File Input"
description: "Send documents and files alongside text in Chat Completions using the OpenAI-compatible SDK."
---

The [Chat Completions API](/en/sdk/openai-compat/chat-completions) supports file inputs via the OpenAI SDK. You can attach documents, images, and other files for models to analyze using `file.file_id` (pre-uploaded), inline `file.file_data` (base64), or `file.file_url` (public URL).

---

## Upload a File

First, upload your file using the `/v1/files` endpoint with `purpose=user_data`:

<CodeGroup>

```bash curl
curl https://api.neosantara.xyz/v1/files \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -F purpose="user_data" \
  -F file="@document.pdf"
```

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
import fs from 'fs';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1'
});

const file = await client.files.create({
  purpose: 'user_data',
  file: fs.createReadStream('document.pdf')
});
console.log(file.id); // file-abc123
```

```python Python (OpenAI SDK)
from openai import OpenAI
client = OpenAI(
    api_key=os.environ["NAI_API_KEY"],
    base_url="https://api.neosantara.xyz/v1"
)

file = client.files.create(
    purpose="user_data",
    file=open("document.pdf", "rb")
)
print(file.id)  # file-abc123
```

</CodeGroup>

## Using `file.file_id` (Pre-uploaded)

Once uploaded, reference the file by its `file_id`:

<CodeGroup>

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});

const chat = await client.chat.completions.create({
  model: 'claude-opus-4-6',
  messages: [
    {
      role: 'user',
      content: [
        { type: 'file', file: { file_id: 'file-abc123' } },
        { type: 'text', text: 'What does this file contain?' },
      ],
    },
  ],
});
console.log(chat.choices[0].message.content);
```

```python Python (OpenAI SDK)
from openai import OpenAI
client = OpenAI(
    api_key=os.environ["NAI_API_KEY"],
    base_url="https://api.neosantara.xyz/v1"
)

chat = client.chat.completions.create(
    model="claude-opus-4-6",
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "file", "file": {"file_id": "file-abc123"}},
                {"type": "text", "text": "What does this file contain?"},
            ],
        }
    ],
)
print(chat.choices[0].message.content)
```

```bash curl
curl https://api.neosantara.xyz/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -d '{
    "model": "claude-opus-4-6",
    "messages": [{
      "role": "user",
      "content": [
        { "type": "file", "file": { "file_id": "file-abc123" } },
        { "type": "text", "text": "What does this file contain?" }
      ]
    }]
  }'
```

</CodeGroup>

## Using `file.file_data` (Inline Base64)

<CodeGroup>

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});

const fileContent = Buffer.from('Hello, this is a test file.').toString('base64');

const chat = await client.chat.completions.create({
  model: 'claude-opus-4-6',
  messages: [
    {
      role: 'user',
      content: [
        { type: 'file', file: { file_data: fileContent, filename: 'hello.txt' } },
        { type: 'text', text: 'Read this file.' },
      ],
    },
  ],
});
console.log(chat.choices[0].message.content);
```

```bash curl
curl https://api.neosantara.xyz/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -d '{
    "model": "claude-opus-4-6",
    "messages": [{
      "role": "user",
      "content": [
        { "type": "file", "file": { "file_data": "SGVsbG8sIHRoaXMgaXMgYSB0ZXN0IGZpbGUu", "filename": "hello.txt" } },
        { "type": "text", "text": "Read this file." }
      ]
    }]
  }'
```

</CodeGroup>

## Using `file.file_url` (Public URL)

<CodeGroup>

```typescript Node.js (OpenAI SDK)
import OpenAI from 'openai';
const client = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});

const chat = await client.chat.completions.create({
  model: 'claude-opus-4-6',
  messages: [
    {
      role: 'user',
      content: [
        { type: 'file', file_url: { url: 'https://example.com/document.pdf' } },
        { type: 'text', text: 'Summarize this document.' },
      ],
    },
  ],
});
console.log(chat.choices[0].message.content);
```

```bash curl
curl https://api.neosantara.xyz/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $NAI_API_KEY" \
  -d '{
    "model": "claude-opus-4-6",
    "messages": [{
      "role": "user",
      "content": [
        { "type": "file", "file_url": { "url": "https://example.com/document.pdf" } },
        { "type": "text", "text": "Summarize this document." }
      ]
    }]
  }'
```

</CodeGroup>

<Note>
  File upload size is limited to 100 MB. For the Responses API, see [File Input](/en/sdk/responses-api/file-input). For the Anthropic-compatible endpoint, see [File Attachments](/en/sdk/anthropic-compat/file-attachments).
</Note>
