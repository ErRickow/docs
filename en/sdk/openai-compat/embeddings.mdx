Generate vector embeddings from input text for semantic search, similarity matching, and retrieval-augmented generation (RAG). Neosantara provides an OpenAI-compatible embeddings endpoint that works with popular models and libraries.

**Endpoint**

`POST /v1/embeddings`

### Example request

Use the standard OpenAI SDK and point it to the Neosantara AI gateway.

<CodeGroup>

```typescript embeddings.ts
import OpenAI from 'openai';
 
const openai = new OpenAI({
  apiKey: process.env.NAI_API_KEY,
  baseURL: 'https://api.neosantara.xyz/v1',
});
 
const response = await openai.embeddings.create({
  model: 'nusa-embedding-0001',
  input: 'Sunny day at the beach',
});
 
console.log(response.data[0].embedding);
```

```python embeddings.py
import os
from openai import OpenAI
 
client = OpenAI(
    api_key=os.getenv("NAI_API_KEY"),
    base_url="https://api.neosantara.xyz/v1",
)
 
response = client.embeddings.create(
    model="nusa-embedding-0001",
    input="Sunny day at the beach",
)
 
print(response.data[0].embedding)
```

</CodeGroup>

---

### Response format

The response returns a list of embedding objects conforming to the OpenAI specification.

```json
{
  "object": "list",
  "data": [
    {
      "object": "embedding",
      "index": 0,
      "embedding": [-0.0038, 0.021, 0.045, ...]
    }
  ],
  "model": "nusa-embedding-0001",
  "usage": {
    "prompt_tokens": 6,
    "total_tokens": 6
  }
}
```
