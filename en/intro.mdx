Neosantara is a cutting-edge AI gateway designed to simplify how developers interact with the world's most powerful language and multimodal models. By providing a single point of entry, we eliminate the complexity of managing multiple API keys, diverse SDKs, and inconsistent response formats.

## Key Features

*   **Unified API Key**: Access OpenAI, Anthropic, Google, and Open-Source models with one credential.
*   **Triple-SDK Compatibility**: Full native support for the **OpenAI SDK**, **Anthropic SDK**, and our high-performance **Responses API**.
*   **Advanced Reasoning**: Standardized support for chain-of-thought processing and native reasoning extraction.
*   **Regional Performance**: Infrastructure optimized for low-latency access and regional context.
