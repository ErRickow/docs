### Model Pricing

Pricing for NeosantaraAI models is based on a per-token usage model for text models and per-image for generation models. The cost varies by model complexity and capabilities.

**Indonesian-Optimized Models:**
- `nusantara-base`: Rp 300/1,500 per 1M tokens (input/output)
- `archipelago-70b`: Rp 4,710/36,543 per 1M tokens (input/output)
- `garda-beta-mini`: Rp 2,500/10,499 per 1M tokens (input/output)
- `luminai`: Free

**Image Generation:**
- `neosantara-gen-2045`: Rp 2,000 (1024x1024), Rp 1,200 (512x512), Rp 500 (256x256)

Refer to our [pricing documentation](/en/about/token-credits) for complete pricing details and usage tiers.

When you make an API request, the response will output both input and output token counts as part of the reported `usage` metrics.

## Model Capabilities

Here is a list of the main capabilities supported by Neosantara AI models:

| Capability | Description |
|------------|-------------|
| text_generation | The ability of the model to generate human-like text responses |
| function_calling | The ability of the model to call external functions |
| json_mode | The ability of the model to generate output in structured JSON format |
| web_search | The ability of the model to search for information from the web |
| vision | The ability of the model to analyze and understand content from images |
| image_generation | The ability of the model to create images from text prompts |
| embedding | The ability of the model to convert text into numerical vector representations |
