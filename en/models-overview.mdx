### Model Comparison Table

To help you choose the right model for your needs, we've compiled a table comparing the key features and capabilities of some core NeosantaraAI models.

| Feature             | Nusantara-Base                                                       | Neosantara-Gen-2045                                               | Garda-Beta-Mini                                                |
| :------------------ | :------------------------------------------------------------------- | :---------------------------------------------------------------- | :--------------------------------------------------------- |
| **Description** | General-purpose model with balanced performance and speed.           | Our main image generation model.                                  | Fast and efficient model for Indonesian language.          |
| **Strengths** | Versatile, good for most common tasks.                               | High-quality image generation, creative content.                  | Optimized for Indonesian language understanding.           |
| **Multilingual** | Yes                                                                  | Yes (for prompts)                                                 | Yes                                                        |
| **Vision** | Yes (via integration with underlying models like Gemini-2.0-Flash)   | No (Generates images, does not analyze them)                      | No                                                         |
| **Function Calling**| Yes                                                                  | No                           | Yes                                                        |
| **JSON Mode** | Yes                                                                  | No (Not for structured text output)                               | Yes                                                        |
| **Context Window** | 64,000 tokens                                                        | 0 (Not applicable for image generation)                           | 8,192 tokens                                               |
| **Max Output** | 2,048 tokens                                                         | 0 (Output is an image, not text tokens)                           | 2,048 tokens                                               |
| **Training Data Cut-off** | Varies by underlying provider | Varies by underlying provider        | Varies by underlying provider |
| **Comparative Latency** | Fast                                                                 | Fast                                                              | Fastest                                                    |

### Model Pricing

Pricing for NeosantaraAI models is based on a per-token usage model. The cost varies by model and usage tier.

Refer to our [pricing documentation](https://neosantara.xyz/pricing) for current prices and usage tiers.

When you make an API request, the response will output both input and output token counts as part of the reported `usage` metrics.
